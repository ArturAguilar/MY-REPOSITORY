Gradientes

O CSS3 permite criar transições suaves entre duas ou mais cores usando gradientes. Isso evita a necessidade de imagens e melhora a performance do site.

Tipos de Gradientes

Gradiente Linear (linear-gradient)

Transição de cores em linha reta.

Sintaxe:

background: linear-gradient(direção, cor1, cor2, ...);

Exemplo:

div {
background: linear-gradient(to right, red, blue);
}

Direções Possíveis:

Direção Sintaxe
De cima para baixo (padrão) linear-gradient(red, blue)
Da esquerda para a direita linear-gradient(to right, red, blue)
Diagonal linear-gradient(to bottom right, red, blue)

Exemplo Avançado:

div {
background: linear-gradient(45deg, red, yellow, green);
}

Gradiente Radial (radial-gradient)

Transição circular ou elíptica.

Sintaxe:

background: radial-gradient(formato tamanho, cor1, cor2, ...);

Exemplo:

div {
background: radial-gradient(circle, red, yellow, green);
}

Tipos de Formato:

Formato Sintaxe
Circular (padrão) radial-gradient(circle, red, blue)
Elíptico lradial-gradient(ellipse, red, blue)

Exemplo com Foco Ajustado:

div {
background: radial-gradient(circle at 30% 70%, red, blue);
}

Gradiente Cônico (conic-gradient)

Gradiente ao redor de um ponto central.

Sintaxe:

background: conic-gradient(cor1 grau, cor2 grau, ...);

Exemplo:

div {
background: conic-gradient(red 0deg, yellow 90deg, green 180deg, blue 270deg);
}

Criando um "Gráfico de Pizza":

div {
background: conic-gradient(red 0deg 90deg, yellow 90deg 180deg, green 180deg 270deg, blue 270deg 360deg);
}

Exemplo Completo com Gradientes

body {
background: linear-gradient(to bottom right,
#ff7e5f, #feb47b);
}

div {
width: 300px;
height: 300px;
background: radial-gradient(circle, blue, white);
border-radius: 50%;
}

h1 {
background: conic-gradient(red, yellow, green, blue);
-webkit-background-clip: text;
color: transparent;
}
