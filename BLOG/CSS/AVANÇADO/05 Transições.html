Transições

Uma transição no CSS3 permite que você defina o tempo e o efeito de uma mudança nas propriedades de um elemento. Quando um estado do elemento muda, a transição acontece ao longo de um intervalo de tempo.

Propriedade Comum:

transition

A propriedade transition é usada para especificar o que deve acontecer durante a transição de um elemento. Ela envolve quatro aspectos principais:

  • transition-property: Especifica qual propriedade será animada.
  • transition-duration: Define o tempo que a transição deve levar.
  • transition-timing-function: Define a aceleração da transição (como ela vai acelerar ou desacelerar durante o tempo).
  • transition-delay: Define o tempo de espera antes da transição começar.

Sintaxe de Transição:

element {
transition: property duration timing-function delay;
}

Onde:

  • property: A propriedade CSS que será animada (ex: background-color, height, etc.).
  • duration: O tempo que a transição levará (ex: 1s, 500ms).
  • timing-function: Como a transição será acelerada/desacelerada (ease, linear, etc.).
  • delay: Tempo de espera antes da transição começar.

Exemplos de Transições:

Exemplo Simples:

div {
width: 100px;
height: 100px;
background-color: red;
transition: background-color 0.5s ease-in-out;
}

div:hover {
background-color: blue;
}

O que acontece: Quando você passa o mouse sobre o div, a cor de fundo muda de vermelho para azul ao longo de 0.5 segundos.

Transição em Várias Propriedades:

div {
width: 100px;
height: 100px;
background-color: red;
transition: width 1s, height 1s, background-color 0.5s;
}

div:hover {
width: 200px;
height: 200px;
background-color: blue;
}

O que acontece: Quando o mouse passa sobre o div, o tamanho e a cor do fundo mudam ao mesmo tempo, mas com tempos de transição diferentes.

Propriedades Comuns Usadas em Transições:

  • background-color: Altera a cor de fundo de um elemento.
  • width / height: Alteram o tamanho de um elemento.
  • opacity: Altera a opacidade (transparência) de um elemento.
  • transform: Permite que você mude a posição, rotação ou escala de um elemento.
  • color: Altera a cor do texto.

Funções de Tempo para Transições (transition-timing-function)

A função de tempo descreve como a transição será acelerada ou desacelerada. Aqui estão algumas opções populares:

  • ease: Inicia devagar, acelera no meio e desacelera no final.
  • linear: A transição acontece a uma velocidade constante.
  • ease-in: Inicia devagar e depois acelera.
  • ease-out: Inicia rápido e depois desacelera.
  • ease-in-out: Começa devagar, acelera e desacelera no final.

Exemplo de Função de Tempo:

div {
transition: transform 1s ease-in-out;
}

div:hover {
transform: scale(1.5);
}

O que acontece: Quando você passa o mouse sobre o div, ele aumenta de tamanho com um efeito suave, começando devagar e acelerando no meio.

Função de Atraso para Transições (transition-delay)

Você também pode adicionar um atraso antes da transição começar usando a propriedade transition-delay.

div {
transition: background-color 1s ease-in-out 0.5s;
}

div:hover {
background-color: blue;
}

O que acontece: A mudança de cor de fundo para azul acontece com um atraso de 0.5 segundos depois de o mouse passar sobre o div.
