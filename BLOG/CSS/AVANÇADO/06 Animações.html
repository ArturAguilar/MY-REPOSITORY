Animações

As animações no CSS3 permitem criar movimentos e efeitos visuais avançados, onde você pode definir diferentes etapas da animação usando keyframes (quadros-chave). Ao contrário das transições, que exigem um evento para começar, as animações no CSS3 são automáticas e podem se repetir.

Propriedade Comum:

animation

A propriedade animation é usada para definir o que

  • acontecerá durante a animação, controlando:

  • animation-name: O nome da animação, que faz referência a um conjunto de keyframes.
  • animation-duration: O tempo total que a animação levará para ser executada.
  • animation-timing-function: Define a aceleração da animação (ex: ease, linear).
  • animation-delay: O tempo de espera antes da animação começar.
  • animation-iteration-count: Define o número de vezes que a animação será executada.
  • animation-direction: Define se a animação deve ser repetida em ordem normal ou reversa.
  • animation-fill-mode: Define o comportamento da animação antes ou depois de ela ser executada.
  • animation-play-state: Controla se a animação está em pausa ou em execução.

Sintaxe de Animação:

element {
animation: name duration timing-function delay iteration-count direction fill-mode;
}

Onde:

  • name: O nome da animação definida pelos @keyframes.
  • duration: O tempo total da animação (ex: 1s, 500ms).
  • timing-function: O tipo de aceleração/desaceleração (ex: ease, linear).
  • delay: Tempo de espera antes da animação começar (ex: 1s).
  • iteration-count: O número de repetições da animação (ex: infinite para repetição infinita).
  • direction: Define se a animação deve rodar de forma normal ou reversa.
  • fill-mode: Controla o comportamento da animação antes ou depois de ela ser executada (ex: forwards, backwards).

Exemplo de Animação Simples:

@keyframes slide {
from {
left: 0;
}
to {
left: 200px;
}
}

div {
position: relative;
animation: slide 2s ease-in-out;
}

O que acontece: Um div se move da posição inicial (esquerda 0) para a posição de 200px em 2 segundos com uma aceleração suave.

Keyframes:

O @keyframes é utilizado para definir os estágios da animação. Ele permite que você defina as propriedades que devem ser alteradas durante a animação e em que pontos da execução da animação isso deve acontecer.

Exemplo de @keyframes:

@keyframes example {
0% {
background-color: red;
}
50% {
background-color: yellow;
}
100% {
background-color: green;
} }


div {
animation: example 3s infinite;
}

O que acontece: A animação muda a cor de fundo do div de vermelho para amarelo e, finalmente, para verde, repetindo o ciclo indefinidamente.

Controlando o Número de Repetições da Animação:

A animação pode ser repetida um número determinado de vezes ou indefinidamente, utilizando a propriedade animation-iteration-count.

Exemplo de Repetição:

div {
animation: example 3s infinite;
}

O que acontece: A animação continuará a se repetir para sempre (infinite), alterando as cores.

Outro Exemplo com Número de Repetições:

div {
animation: example 3s 3;
}

O que acontece: A animação será executada 3 vezes e, depois, será interrompida.

Direção da Animação:

Você pode definir a direção da animação, controlando se a animação deve ser executada na sequência normal ou reversa.

  • normal: A animação segue a direção definida (padrão).
  • reverse: A animação vai para trás.
  • alternate: A animação alterna entre as direções, fazendo um ciclo para frente e outro para trás.
  • alternate-reverse: A animação alterna as direções, mas começa ao contrário.

Exemplo com Direção:

div {
animation: example 3s alternate;
}

O que acontece: A animação alterna entre as direções durante a execução.

Atraso de Animação:

A animação pode começar após um atraso utilizando a propriedade animation-delay.

div {
animation: example 3s 1s;
}

O que acontece: A animação começará 1 segundo depois de o div ser carregado, e durará 3 segundos.

Funções de Tempo para Animação (animation-timing-function):

As funções de tempo controlam a aceleração da animação. Aqui estão algumas opções:

  • ease: Inicia devagar, acelera no meio e desacelera no final.
  • linear: A animação acontece a uma velocidade constante.
  • ease-in: Inicia devagar e depois acelera.
  • ease-out: Começa rápido e depois desacelera.
  • ease-in-out: Começa devagar, acelera e desacelera no final.
