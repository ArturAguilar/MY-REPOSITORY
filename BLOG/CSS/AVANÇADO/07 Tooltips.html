Tooltips

Tooltips são pequenas caixas de texto que aparecem quando o usuário passa o mouse sobre um elemento. Elas são úteis para exibir informações adicionais, como descrições, dicas ou detalhes de uma ação, de maneira discreta.

Como Criar um Tooltip com CSS:

Para criar tooltips em CSS, você pode usar a combinação das propriedades position, visibility, opacity e transition.

Estrutura Básica do Tooltip:

<div class="tooltip">Passe o mouse aqui
<span class="tooltiptext">Texto do tooltip</span>
</div>

  • tooltip: A classe principal que será aplicada ao elemento que exibe o tooltip.
  • tooltiptext: A classe que define o texto exibido dentro do tooltip.

Exemplo CSS:

.tooltip {
position: relative;
display: inline-block;
}

.tooltip .tooltiptext {
visibility: hidden;
width: 120px;
background-color: black;
color: #fff;
text-align: center;
border-radius: 5px;
padding: 5px 0;

/* Posição do tooltip */
position: absolute;
z-index: 1;
bottom: 125%; /* Coloca o tooltip acima do elemento */
left: 50%;
margin-left: -60px; /* Centraliza o tooltip */


/* Efeito de transição */
opacity: 0;
transition: opacity 0.3s;
}

/* Mostrar o tooltip quando o usuário passar o mouse */
.tooltip:hover .tooltiptext {
visibility: visible;
opacity: 1;
}

Explicação do Código:

  • .tooltip:
    • position: relative: A posição relativa permite que o tooltiptext seja posicionado em relação a este elemento.
    • display: inline-block: Garante que o elemento se comporte como um bloco, mas permaneça em linha com outros elementos.
  • tooltiptext:
    • visibility: hidden: Inicialmente, o tooltip está invisível.
    • position: absolute: O tooltiptext é posicionado de forma absoluta em relação ao seu elemento pai.
    • bottom: 125%: Coloca o tooltip 125% acima do elemento pai.
    • left: 50% e margin-left: -60px: Centraliza o tooltip em relação ao seu elemento pai.
    • opacity: 0: O tooltip começa invisível.
    • transition: opacity 0.3s: Define uma transição suave quando a opacidade muda.
  • :hover:
    • Quando o usuário passa o mouse sobre o elemento com a classe tooltip, o tooltip se torna visível devido a visibility: visible e opacity: 1.

Personalização de Tooltips:

Você pode personalizar os tooltips com diferentes cores, fontes, bordas, animações e mais. Aqui estão alguns exemplos de personalização:

Tooltip com Background Color e Borda:

.tooltip .tooltiptext {
background-color: blue;
border: 1px solid #f1f1f1;
}

Tooltip com Animação de Desaparecimento:

.tooltip .tooltiptext {
opacity: 0;
transition: opacity 0.5s ease-in-out;
}

.tooltip:hover .tooltiptext {
opacity: 1;
}
