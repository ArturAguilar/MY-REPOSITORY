object-fit e object-position

O CSS3 introduziu propriedades como object-fit e object-position para manipular imagens e vídeos de forma mais precisa dentro de contêineres.

O que é object-fit?

A propriedade object-fit define como uma imagem ou um vídeo deve se ajustar dentro do seu contêiner. Ela funciona em elementos img e video.

✅ Exemplo de Uso

img {
width: 300px;
height: 200px;
object-fit: cover;
}

Principais Valores de object-fit

Valor Descrição
fill (default) A imagem se estica para ocupar todo o espaço do contêiner, podendo distorcer.
contain A imagem é redimensionada para caber dentro do contêiner sem cortar nada. Pode haver espaços vazios.
cover A imagem cobre todo o contêiner, podendo cortar partes dela.
none A imagem mantém seu tamanho original, podendo ultrapassar o contêiner.
scale-down Usa o menor tamanho possível entre none e contain.

✅ Comparação Visual

  • cover: Mantém a proporção e cobre todo o espaço disponível.
  • contain: Mantém a proporção, mas pode deixar espaços vazios.
  • fill: Estica a imagem para caber no contêiner, distorcendo-a.

O que é object-position?

A propriedade object-position permite controlar a posição da imagem dentro do contêiner quando object-fit: cover ou object-fit: contain estão sendo usados.

✅ Exemplo de Uso

img {
width: 300px;
height: 200px;
object-fit: cover;
object-position: top right;
}

Principais Valores de object-position

Valor Descrição
center center (default) Centraliza a imagem.
top left Alinha a imagem no canto superior esquerdo.
top right Alinha a imagem no canto superior direito.
bottom left Alinha a imagem no canto inferior esquerdo.
bottom right Alinha a imagem no canto inferior direito.
50% 50% Define posições personalizadas usando porcentagens ou pixels.

✅ Ajustando Precisamente a Posição

img {
object-fit: cover;
object-position: 75% 25%;
}

  • 75% 25% move a imagem mais para a direita e um pouco para cima.
