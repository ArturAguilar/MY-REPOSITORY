Masking

O CSS Masking permite esconder ou revelar partes de um elemento usando máscaras opacas e transparentes. Isso é útil para criar efeitos visuais avançados, como gradientes de transparência e cortes personalizados em imagens.

Propriedade mask-image

Define uma máscara que controla a visibilidade de partes de um elemento. Pode ser aplicada com imagens ou gradientes.

Exemplo de Uso: Máscara com Imagem

div {
width: 300px;
height: 200px;
background: url("paisagem.jpg");
mask-image: url("mascara.png");
-webkit-mask-image: url("mascara.png"); /* Compatibilidade com WebKit */
}

  • A máscara (mascara.png) define quais partes da paisagem.jpg serão visíveis.
  • Áreas brancas na máscara ficam visíveis, enquanto áreas pretas ficam ocultas.

Propriedade mask-size

Define o tamanho da máscara, semelhante ao background-size.

Exemplo de mask-size

div {
mask-image: url("mascara.png");
mask-size: contain;
-webkit-mask-size: contain;
}

  • contain: ajusta a máscara para caber no elemento sem cortar.
  • cover: faz a máscara cobrir o elemento inteiro.

mask-image com Gradiente

Podemos usar um gradiente como máscara para criar um efeito de transição suave.

Exemplo de Máscara com Gradiente

div {
width: 300px;
height: 200px;
background: url("paisagem.jpg");
mask-image: linear-gradient(to bottom, rgba(0,0,0,1), rgba(0,0,0,0));
-webkit-mask-image: linear-gradient(to bottom, rgba(0,0,0,1), rgba(0,0,0,0));
}

  • A imagem desaparece gradualmente de cima para baixo.
  • Valores escuros mantêm a área visível, e áreas transparentes ocultam.

Propriedade mask-position

Define a posição da máscara dentro do elemento.

Exemplo de mask-position

div {
mask-image: url("mascara.png");
mask-position: center;
-webkit-mask-position: center;
}

Alinha a máscara ao centro do elemento.

Valores comuns:

  • A máscara (mascara.png) define quais partes da paisagem.jpg serão visíveis.
  • Áreas brancas na máscara ficam visíveis, enquanto áreas pretas ficam ocultas.

Propriedade mask-repeat

Controla se a máscara será repetida no elemento.

Exemplo de mask-repeat

div {
mask-image: url("mascara.png");
mask-repeat: repeat-x;
-webkit-mask-repeat: repeat-x;
}

Repete a máscara horizontalmente.

Valores comuns:

  • no-repeat
  • repeat
  • repeat-x
  • repeat-y

Propriedade mask-composite

Define como múltiplas máscaras interagem umas com as outras.

Exemplo de mask-composite

div {
mask-image: url("mascara1.png"), url("mascara2.png");
mask-composite: exclude;
-webkit-mask-composite: exclude;
}

exclude: remove a área onde as máscaras se sobrepõem.

Outros valores:

  • add: soma as máscaras.
  • subtract: subtrai a segunda máscara da primeira.
