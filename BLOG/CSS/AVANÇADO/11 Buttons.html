Buttons

Os botões no CSS podem ser personalizados para melhorar a aparência e a usabilidade. Podemos usar cores, sombras, bordas arredondadas, efeitos de hover e transições para criar um design moderno e interativo.

Estilizando um Botão Básico

A propriedade background-color define a cor de fundo do botão, enquanto color define a cor do texto.

✅ Exemplo de Botão Simples

button {
background-color: #d70000; /* Cor de fundo */
color: white; /* Cor do texto */
padding: 10px 20px; /* Espaçamento interno */
border: none; /* Sem borda */
cursor: pointer; /* Cursor ao passar o mouse */
font-size: 16px; /* Tamanho da fonte */
}

  • padding adiciona espaçamento interno para tornar o botão maior.
  • cursor: pointer; muda o cursor ao passar o mouse.

Botão com Bordas Arredondadas

A propriedade border-radius permite criar botões com cantos arredondados.

✅ Exemplo de Botão Arredondado

button {
background-color: #d70000;
color: white;
padding: 10px 20px;
border: none;
border-radius: 20px; /* Borda arredondada */
cursor: pointer;
}

  • Quanto maior o valor de border-radius, mais arredondado o botão fica.

Efeito Hover no Botão

O :hover permite adicionar um efeito quando o usuário passa o mouse sobre o botão.

✅ Exemplo de Hover

button:hover {
background-color: #a00000; /* Cor mais escura ao passar o mouse */
}

  • Isso melhora a experiência do usuário ao indicar que o botão é clicável.

Adicionando Sombra ao Botão

A propriedade box-shadow cria um efeito de sombra ao redor do botão.

button {
background-color: #d70000;
color: white;
padding: 10px 20px;
border: none;
border-radius: 5px;
box-shadow: 3px 3px 10px rgba(0, 0, 0, 0.3);
}

  • O box-shadow cria uma sombra sutil para dar um efeito 3D.

Botão com Animação de Pressionado (:active)

Podemos usar :active para criar um efeito de "pressionado" quando o botão for clicado.

✅ Exemplo de Animação ao Clicar

button:active {
transform: scale(0.95); /* Reduz o tamanho ligeiramente */
box-shadow: 1px 1px 5px rgba(0, 0, 0, 0.2);
}

  • transform: scale(0.95) reduz ligeiramente o tamanho do botão ao clicar.

Adicionando Ícones aos Botões

Podemos incluir ícones nos botões usando Font Awesome ou o elemento .

✅ Exemplo de Botão com Ícone

<button>
<i class="fa fa-check"></i> Confirmar
</button>

  • Para usar os ícones, adicione a biblioteca Font Awesome ao seu projeto.

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0/css/all.min.css">

Criando um Botão Transparente

O botão pode ter um fundo transparente usando background: transparent e border.

✅ Exemplo de Botão Transparente

button {
background: transparent;
color: #d70000;
border: 2px solid #d70000;
padding: 10px 20px;
cursor: pointer;
}

button:hover {
background-color: #d70000;
color: white;
}

  • Um botão transparente com borda fica sólido ao passar o mouse.

Criando um Botão com Gradiente

Podemos usar background: linear-gradient para criar um efeito gradiente.

✅ Exemplo de Botão com Gradiente

button {
background: linear-gradient(to right, #ff0000, #d70000);
color: white;
padding: 10px 20px;
border: none;
border-radius: 5px;
}

  • Gradientes criam um efeito mais moderno e chamativo.
