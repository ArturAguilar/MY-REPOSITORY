Paginação

A paginação é essencial para dividir grandes conjuntos de dados ou páginas de um site. No CSS, podemos criar botões de próxima página, página anterior e números de páginas estilizados.

Criando uma Estrutura de Paginação

A estrutura básica pode ser feita com uma lista (ul e li).

Exemplo de Estrutura HTML

<ul class="pagination">
<li><a href="#">«</a></li> <!-- Página Anterior -->
<li><a href="#">1</a></li>
<li><a href="#">2</a></li>
<li><a href="#">3</a></li>
<li><a href="#">»</a></li> <!-- Próxima Página -->
</ul>

  • « e » são símbolos para "Anterior" e "Próximo".
  • Cada li representa um número de página.

Estilizando a Paginação com CSS

Agora aplicamos estilos para criar um design mais amigável.

✅ Exemplo de CSS

.pagination {
display: flex;
list-style: none;
padding: 0;
}

.pagination li {
margin: 5px;
}

.pagination a {
text-decoration: none;
background: #d70000; /* Cor principal */
color: white;
padding: 8px 12px;
border-radius: 5px;
transition: 0.3s;
}

.pagination a:hover {
background: #a00000; /* Efeito ao passar o mouse */
}

  • display: flex; alinha os botões horizontalmente.
  • border-radius arredonda os cantos.
  • transition: 0.3s; cria um efeito suave no hover.

Destacando a Página Ativa

Podemos adicionar uma classe active para indicar a página atual.

✅ Exemplo de CSS

.pagination .active a {
background: black;
color: white;
}

✅ Aplicação no HTML

<ul class="pagination">
<li><a href="#">«</a></li>
<li><a href="#">1</a></li>
<li class="active"><a href="#">2</a></li> <!-- Página Atual -->
<li><a href="#">3</a></li>
<li><a href="#">»</a></li>
</ul>

A página ativa fica destacada com fundo preto.

Paginação Redonda

Podemos criar botões circulares alterando o border-radius.

✅ Exemplo de CSS

.pagination a {
border-radius: 50%;
width: 35px;
height: 35px;
display: flex;
align-items: center;
justify-content: center;
}

Cada botão terá formato redondo.

Paginação Responsiva

Se houver muitas páginas, podemos adicionar "..." para ocultar algumas delas.

✅ Exemplo de CSS

<ul class="pagination">
<li><a href="#">«</a></li>
<li><a href="#">1</a></li>
<li><a href="#">2</a></li>
<li>...</li>
<li><a href="#">9</a></li>
<li><a href="#">10</a></li>
<li><a href="#">»</a></li>
</ul>

Isso evita que a lista de páginas fique muito longa.
