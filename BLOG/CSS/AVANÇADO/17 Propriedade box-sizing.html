Propriedade box-sizing

A propriedade box-sizing define como a largura e altura de um elemento devem ser calculadas: incluindo ou excluindo o padding e a borda.

Problema sem box-sizing

Por padrão, quando definimos width e height para um elemento, o padding e a borda são adicionados ao tamanho total do elemento.

Sem box-sizing: border-box;

.div1 {
width: 200px;
padding: 20px;
border: 5px solid black;
}

O tamanho final do elemento será:

Largura total = 200px (width) + 20px (padding esquerdo) + 20px (padding direito) + 5px (borda esquerda) + 5px (borda direita)
Largura total = 250px

O elemento fica maior do que o esperado!

Solução com box-sizing: border-box;

Ao usar box-sizing: border-box;, o padding e a borda são incluídos na largura e altura especificadas.

Com box-sizing: border-box;

.div2 {
width: 200px;
padding: 20px;
border: 5px solid black;
box-sizing: border-box;
}

Agora o tamanho final será exatamente 200px!

  • O padding e a borda não aumentam o tamanho total do elemento.
  • Isso facilita o controle do layout e evita cálculos complexos de dimensionamento.

Valores da Propriedade box-sizing

Valor Descrição
content-box (Padrão) A largura e altura excluem padding e borda.
border-box A largura e altura incluem padding e borda.

Exemplo Completo

.box {
width: 300px;
height: 100px;
padding: 20px;
border: 5px solid black;
}

.box1 {
box-sizing: content-box;
background-color: lightblue;
}

.box2 {
box-sizing: border-box;
background-color: lightgreen;
}

<div class="box box1">content-box</div>
<div class="box box2">border-box</div>

  • O primeiro bloco (content-box) será maior que 300px, pois padding e borda aumentam o tamanho total.
  • O segundo bloco (border-box) terá exatamente 300px, pois padding e borda são considerados dentro desse valor.
