O que é padding?

A propriedade padding define o espaço interno entre o conteúdo de um elemento e sua borda.

Sintaxe

elemento {
padding: valor;
}

O valor pode ser em pixels (px), porcentagem (%), em, rem e outras unidades.

Exemplo Simples

HTML

<div class="caixa">Espaçamento interno</div>

CSS

.caixa {
margin: 20px;
background-color: lightblue;
}

Esse código adiciona 20px de espaçamento interno ao redor do conteúdo da div.

Definindo padding Individualmente

Você pode definir valores diferentes para cada lado:

elemento {
padding-top: 10px;
padding-right: 15px;
padding-bottom: 20px;
padding-left: 25px;
}

Ou usar shorthand (atalho):

padding: 10px 15px 20px 25px;
/* topo, direita, baixo, esquerda */

Outras variações:

padding: 10px 20px; /* topo e baixo = 10px, esquerda e direita = 20px */
padding: 15px; /* todos os lados = 15px */

padding com width e height

Por padrão, o padding aumenta o tamanho total do elemento.

Problema

.caixa {
width: 200px;
padding: 20px;
}

O tamanho real da .caixa será 240px (200px de largura + 20px de padding em cada lado).

Corrigindo com box-sizing: border-box;

Para manter o width e height fixos, use:

.caixa {
width: 200px;
padding: 20px;
box-sizing: border-box;
}

Agora o padding não altera o tamanho total do elemento.
