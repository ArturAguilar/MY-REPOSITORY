Outline

O contorno é uma linha desenhada ao redor de um elemento, fora da borda. Diferente da borda (border), o contorno não afeta o tamanho do elemento nem o fluxo do layout. É comumente utilizado para destacar elementos, especialmente durante interações como foco em formulários.

Propriedades Principais do Contorno

  • outline-style: Define o estilo da linha do contorno.
  • outline-width: Especifica a largura do contorno.
  • outline-color: Determina a cor do contorno.
  • outline-offset: Controla o espaçamento entre a borda do elemento e o contorno.

outline-style

Esta propriedade define o estilo da linha do contorno e é obrigatória para que o contorno seja exibido.

Valores Comuns:

  • none: Sem contorno.
  • solid: Linha sólida.
  • dotted: Linha pontilhada.
  • dashed: Linha tracejada.
  • double: Linha dupla.
  • groove: Aparência de sulco.
  • ridge: Aparência de relevo.
  • inset: Aparência de inserção.
  • outset: Aparência de projeção.

Exemplo:

button {
outline-style: solid;
}

outline-width

Especifica a largura do contorno.

Valores:

  • Medidas específicas: px, em, etc.
  • Palavras-chave:
    • thin: Fino.
    • medium: Médio (padrão).
    • thick: Espesso.

Exemplo:

button {
outline-style: solid;
outline-width: 2px;
}

outline-color

Define a cor do contorno.

Valores:

  • Nomes de cores: red, blue, etc.
  • Valores hexadecimais: #ff0000, #0000ff, etc.
  • Valores RGB/RGBA: rgb(255, 0, 0), rgba(0, 0, 255, 0.5), etc.
  • Palavras-chave:
    • invert: Inverte a cor em relação ao fundo, garantindo contraste.
    • currentColor: Utiliza a cor atual do elemento.

Exemplo:

button {
outline-style: solid;
outline-width: 2px;
outline-color: #ff0000;
}

outline-offset

Controla o espaço entre a borda do elemento e o contorno.

Valores:

Medidas específicas: px, em, etc.

Exemplo:

button {
outline-style: solid;
outline-width: 2px;
outline-color: #ff0000;
outline-offset: 5px;
}

No exemplo acima, o contorno será desenhado 5px além da borda do botão.

Propriedade Shorthand: outline

Você pode definir todas as propriedades do contorno em uma única declaração usando a propriedade abreviada outline.

Sintaxe:

outline: [outline-width] [outline-style] [outline-color];

Exemplo:

button {
outline: 2px solid #ff0000;
}

Diferenças entre outline e border

  • Espaçamento: O contorno é desenhado fora da borda do elemento e não afeta o layout ou o tamanho do elemento.
  • Arredondamento: O contorno não segue os cantos arredondados definidos por border-radius.
  • Herança: As propriedades de contorno não são herdadas pelos elementos filhos.
