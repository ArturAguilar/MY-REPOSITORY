Propriedade z-index

A propriedade z-index determina a ordem de empilhamento de elementos posicionados em uma página, ou seja, quais elementos aparecem na frente ou atrás de outros quando há sobreposição.

Valores Possíveis:

  • Número inteiro: Pode ser positivo, negativo ou zero. Um valor maior indica que o elemento será exibido acima de elementos com valores menores.
  • auto: Define a ordem de empilhamento padrão do elemento, que geralmente é 0.

Importante: A propriedade z-index só funciona em elementos que têm a propriedade position definida como absolute, relative, fixed ou sticky.

Exemplo:

<!DOCTYPE html>
<html>
<head>
<style>
.container {
position: relative;
width: 200px;
height: 200px;
}

.box1 {
position: absolute;
top: 20px;
left: 20px;
width: 100px;
height: 100px;
background-color: red;
z-index: 1;
}

.box2 {
position: absolute;
top: 40px;
left: 40px;
width: 100px;
height: 100px;
background-color: blue;
z-index: 2;
}

.box3 {
position: absolute;
top: 60px;
left: 60px;
width: 100px;
height: 100px;
background-color: green;
z-index: 3;
}
</style>
</head>
<body>
<div class="container">
<div class="box1">Caixa 1</div>
<div class="box2">Caixa 2</div>
<div class="box3">Caixa 3</div>
</div>
</body>
</html>

Neste exemplo, temos três caixas (box1, box2 e box3) posicionadas de forma absoluta dentro de um contêiner. Cada caixa tem um valor de z-index diferente:

  • box1 tem z-index: 1
  • box2 tem z-index: 2
  • box3 tem z-index: 3

Como resultado, a box3 (verde) será exibida acima da box2 (azul), que, por sua vez, será exibida acima da box1 (vermelha), devido aos valores crescentes de z-index.

Nota: Se dois elementos têm o mesmo valor de z-index, a ordem de empilhamento será determinada pela ordem em que aparecem no DOM (Document Object Model). O elemento que aparece por último no código HTML será exibido acima dos anteriores.

Manipulação do z-index com JavaScript

Você pode alterar dinamicamente o valor de z-index de um elemento usando JavaScript, o que é útil para criar interações dinâmicas na página.

Exemplo:

<!DOCTYPE html>
<html>
<head>
<style>
.box {
position: absolute;
width: 100px;
height: 100px;
background-color: red;
z-index: 1;
}
</style>
</head>
<body>
<div class="box" id="myBox">Caixa</div>
<button onclick="bringToFront()">Trazer para Frente</button>

<script>
function bringToFront() {
var box = document.getElementById("myBox");
box.style.zIndex = "10";
}
</script>
</body>
</html>

Neste exemplo, ao clicar no botão "Trazer para Frente", a função bringToFront é chamada, que altera o valor de z-index da caixa (myBox) para 10, fazendo com que ela seja exibida acima de outros elementos com valores de z-index menores.
