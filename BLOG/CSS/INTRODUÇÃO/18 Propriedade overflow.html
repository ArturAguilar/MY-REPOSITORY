Propriedade overflow

A propriedade overflow define como o navegador deve lidar com o conteúdo que ultrapassa as bordas de um elemento. Ela é aplicável a elementos de bloco com dimensões específicas.

Valores Comuns:

  • visible: Padrão. O conteúdo que excede as dimensões do elemento é exibido além de suas bordas.
  • hidden: O conteúdo excedente é cortado, ficando invisível.
  • scroll: Adiciona barras de rolagem, permitindo que o usuário visualize o conteúdo completo.
  • auto: Adiciona barras de rolagem apenas se necessário.

Exemplo:

<!DOCTYPE html>
<html>
<head>
<style>
.container {
width: 200px;
height: 100px;
background-color: lightblue;
overflow: auto;
}
</style>
</head>
<body>
<div class="container">
<p>Este é um exemplo de texto que é longo o suficiente para causar um overflow no elemento pai.</p>
</div>
</body>
</html>

Neste exemplo, o contêiner tem dimensões fixas e o valor overflow: auto;. Se o conteúdo exceder essas dimensões, barras de rolagem serão adicionadas conforme necessário.

Propriedades overflow-x e overflow-y

Para controlar o overflow horizontal e vertical separadamente, utilizamos as propriedades overflow-x e overflow-y.

Exemplo:

.container {
width: 200px;
height: 100px;
background-color: lightblue;
overflow-x: scroll;
overflow-y: hidden;
}

Aqui, o conteúdo que exceder a largura do contêiner resultará em uma barra de rolagem horizontal, enquanto o conteúdo que exceder a altura será ocultado.
