Propriedade float

A propriedade float permite que um elemento seja posicionado à esquerda ou à direita dentro de seu contêiner, fazendo com que o texto e outros elementos adjacentes fluam ao seu redor.

Valores Comuns:

  • left: O elemento flutua à esquerda do contêiner.
  • right: O elemento flutua à direita do contêiner.
  • none: Padrão. O elemento não flutua; é exibido na ordem normal do documento.

Exemplo:

<!DOCTYPE html>
<html>
<head>
<style>
.box {
width: 200px;
height: 100px;
background-color: lightblue;
margin: 10px;
float: left;
}
</style>
</head>
<body>
<div class="box">Caixa 1</div>
<div class="box">Caixa 2</div>
<div class="box">Caixa 3</div>
<p>Este é um parágrafo que fluirá ao redor das caixas flutuantes.</p>
</body>
</html>

Neste exemplo, três caixas são definidas com a classe box e possuem a propriedade float: left;. Como resultado, elas flutuam à esquerda, permitindo que o parágrafo seguinte flua ao redor delas.

Propriedade clear

A propriedade clear é usada para controlar o comportamento de elementos em relação a elementos flutuantes anteriores. Ela especifica de que lado(s) um elemento não deve permitir elementos flutuantes adjacentes.

Valores Comuns:

  • left: O elemento não permite elementos flutuantes à sua esquerda.
  • right: O elemento não permite elementos flutuantes à sua direita.
  • both: O elemento não permite elementos flutuantes em ambos os lados.
  • none: Padrão. Permite elementos flutuantes em ambos os lados.

Exemplo:

<!DOCTYPE html>
<html>
<head>
<style>
.box {
width: 200px;
height: 100px;
background-color: lightblue;
margin: 10px;
float: left;
}

.clear {
clear: both;
}
</style>
</head>
<body>
<div class="box">Caixa 1</div>
<div class="box">Caixa 2</div>
<div class="box">Caixa 3</div>
<div class="clear"></div>
<p>Este parágrafo será exibido abaixo das caixas flutuantes.</p>
</body>
</html>

Neste exemplo, as caixas flutuam à esquerda, e o elemento

com a classe clear possui a propriedade clear: both;. Isso faz com que o parágrafo seguinte seja exibido abaixo das caixas flutuantes, evitando que o texto flua ao redor delas.

Exemplo Prático: Layout de Imagem com Texto

A combinação das propriedades float e clear é frequentemente usada para criar layouts onde o texto envolve imagens.

Exemplo:

<!DOCTYPE html>
<html>
<head>
<style>
.image {
float: left;
margin-right: 10px;
}

.text {
overflow: hidden;
}
</style>
</head>
<body>
<img src="imagem.jpg" alt="Descrição da Imagem" class="image">
<div class="text">
<p>Este é um exemplo de texto que envolve uma imagem flutuante à esquerda. O uso da propriedade `float` permite que o texto flua ao redor da imagem, criando um layout mais atraente e dinâmico.</p>
<p>Para evitar que elementos subsequentes sejam afetados pelo float, podemos usar a propriedade `clear` ou técnicas como clearfix.</p>
</div>
</body>
</html>

Neste exemplo, a imagem flutua à esquerda com uma margem direita para espaçamento, e o texto flui ao redor dela. A classe text usa overflow: hidden; como uma técnica de clearfix para conter os elementos flutuantes dentro de seu contêiner.
