Transparência de imagens

A transparência de imagens em CSS pode ser ajustada usando a propriedade opacity, a qual controla o nível de opacidade de um elemento. Além disso, é possível criar imagens transparentes diretamente no HTML usando imagens no formato PNG com transparência, ou aplicar efeitos de transparência em imagens com a ajuda de CSS.

Propriedade opacity

Controla a opacidade (transparência) de um elemento, onde 0 é totalmente transparente e 1 é totalmente opaco.

img {
opacity: 0.5;
}

Explicação:

  • Define que a imagem terá 50% de opacidade, ou seja, será 50% transparente.
  • A opacidade pode ser aplicada a qualquer elemento, não apenas imagens.

Transparência em Imagens PNG

Imagens PNG podem ter transparência nativa, o que permite que o fundo ou outras partes da imagem se misturem com o conteúdo da página.

  • Para usar imagens PNG com transparência, basta incluir a imagem normalmente no HTML e ela terá a transparência.

<img src="imagem-transparente.png" alt="Imagem Transparente">

Explicação:

  • As imagens no formato PNG podem ser salvas com um canal alpha, permitindo transparência.

Usando rgba() para Cores Transparentes

A função rgba() permite criar cores com um nível de transparência (opacidade) usando os valores de cor RGB (vermelho, verde, azul) e um valor alfa (opacidade de 0 a 1).

div {
background-color: rgba(255, 0, 0, 0.5);
}

Explicação:

  • O rgba(255, 0, 0, 0.5) define uma cor vermelha com 50% de transparência para o fundo de um div.
  • O valor alfa (0.5) define o nível de transparência, variando de 0 (totalmente transparente) a 1 (totalmente opaco).

Usando hsla() para Cores Transparentes

A função hsla() também pode ser usada para cores com transparência, utilizando valores de matiz, saturação, luminosidade e alfa.

div {
background-color: hsla(120, 100%, 50%, 0.5);
}

Explicação:

  • A cor hsla(120, 100%, 50%, 0.5) define um verde com 50% de transparência.
