Introdução ao Design Responsivo

O Design Responsivo (RWD - Responsive Web Design) permite que os sites se adaptem automaticamente a diferentes tamanhos de tela, proporcionando uma melhor experiência para usuários em celulares, tablets e desktops.

O Que é Design Responsivo?

É uma abordagem de desenvolvimento web que ajusta o layout e os elementos da página conforme o tamanho do dispositivo. Isso evita barras de rolagem horizontais e melhora a legibilidade.

✅ O site se ajusta automaticamente.
✅ Nenhum redirecionamento para uma versão "mobile".
✅ Melhor experiência para todos os dispositivos.

Técnicas essenciais:

✔ Uso de Media Queries
✔ Uso de Unidades Relativas (%, vh, vw, rem, em)
✔ Uso de Layouts Flexíveis (flexbox, grid)
✔ Uso de Imagens Responsivas (max-width: 100%)
