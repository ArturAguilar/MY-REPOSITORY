Grid no Design Responsivo

O CSS Grid é um sistema poderoso para layout responsivo, permitindo criar páginas flexíveis sem precisar de muitos hacks. Ele divide o espaço da tela em colunas e linhas, tornando mais fácil organizar os elementos.

O que é CSS Grid?

O CSS Grid divide a tela em linhas e colunas, permitindo distribuir elementos de forma flexível.
Diferente do Flexbox (que é unidimensional), o Grid é bidimensional, ou seja, trabalha com largura e altura ao mesmo tempo.

Criando um Grid Container

Código Básico

<!DOCTYPE html>
<html lang="pt">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>CSS Grid</title>
<style>
.grid-container {
display: grid;
grid-template-columns: 1fr 1fr;
grid-template-rows: 100px 100px;
gap: 10px;
background-color: #ddd;
padding: 10px;
}

.grid-item {
background-color: #d70000;
color: white;
font-size: 1.5rem;
text-align: center;
display: flex;
justify-content: center;
align-items: center;
border-radius: 8px;
}

</style>
</head>
<body>

<div class="grid-container">
<div class="grid-item">1</div>
<div class="grid-item">2</div>
<div class="grid-item">3</div>
<div class="grid-item">4</div>
</div>

</body>
</html>

Explicação:

  • display: grid; → Ativa o Grid no elemento pai.
  • grid-template-columns: 1fr 1fr; → Cria duas colunas de largura igual.
  • grid-template-rows: 100px 100px; → Define duas linhas com 100px cada.
  • gap: 10px; → Adiciona espaçamento entre os itens.

Criando Layouts com Grid

Definindo Colunas e Linhas Específicas

Podemos definir quantas colunas e linhas quisermos:

.grid-container {
display: grid;
grid-template-columns: 200px 1fr 2fr;
grid-template-rows: 100px auto 150px;
}

Explicação:

  • A 1ª coluna tem 200px.
  • A 2ª coluna ocupa todo o espaço restante (1fr).
  • A 3ª coluna é duas vezes maior que a segunda (2fr).
  • As linhas seguem o mesmo conceito.

Posicionando Itens no Grid

Cada item pode ocupar mais de uma célula usando grid-column e grid-row.

.grid-item:first-child {
grid-column: 1 / span 2; /* Ocupa duas colunas */
grid-row: 1; /* Mantém na primeira linha */
}

Isso cria um layout mais dinâmico!

Ajustando o Alinhamento

Alinhamento dos Itens

Podemos usar justify-items e align-items:

.grid-container {
display: grid;
grid-template-columns: repeat(2, 1fr);
height: 300px;
justify-items: center; /* Alinha horizontalmente */
align-items: center; /* Alinha verticalmente */
}

  • justify-items → Controla o alinhamento horizontal dos itens.
  • align-items → Controla o alinhamento vertical dos itens.

Alinhamento do Grid Como um Todo

Podemos centralizar o grid inteiro na tela:

.grid-container {
display: grid;
width: 400px;
height: 300px;
margin: auto;
justify-content: center;
align-content: center;
}

      justify-content → Centraliza o grid horizontalmente.
      align-content → Centraliza o grid verticalmente.

Grid Responsivo

Podemos tornar o Grid responsivo usando repeat(auto-fit, minmax()):

.grid-container {
display: grid;
grid-template-columns: repeat(auto-fit, minmax(150px, 1fr));
gap: 10px;
}

Isso faz com que as colunas se ajustem automaticamente ao tamanho da tela!

<!DOCTYPE html>
<html lang="pt">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>CSS Grid Responsivo</title>
<style>
.grid-container {
display: grid;
grid-template-columns: repeat(auto-fit, minmax(150px, 1fr));
gap: 10px;
padding: 10px;
}

.grid-item {
background-color: #d70000;
color: white;
font-size: 1.2rem;
text-align: center;
padding: 20px;
border-radius: 8px;
}

</style>
</head>
<body>

<div class="grid-container">
<div class="grid-item">Item 1</div>
<div class="grid-item">Item 2</div>
<div class="grid-item">Item 3</div>
<div class="grid-item">Item 4</div>
<div class="grid-item">Item 5</div>
<div class="grid-item">Item 6</div>
</div>

</body>
</html>

O Layout se adapta automaticamente!
