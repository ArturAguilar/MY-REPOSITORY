Media Queries

As Media Queries no CSS permitem que um site se adapte a diferentes tamanhos de tela, ajustando o layout de acordo com o dispositivo (celulares, tablets, desktops, etc.).

Como funcionam as Media Queries?

Elas verificam características do dispositivo (como largura da tela, resolução e orientação) e aplicam estilos específicos quando essas condições são atendidas.

Sintaxe:

@media (condição) {
/* Regras CSS aqui */
}

Exemplo:

body {
background-color: lightblue;
}

@media (max-width: 600px) {
body {
background-color: lightcoral;
}
}

  • Se a tela for menor que 600px de largura, o fundo muda para vermelho (lightcoral).
  • Se for maior que 600px, o fundo continua azul (lightblue).

Tipos de Condições em Media Queries

Condição Descrição
max-width Aplica estilos se a largura da tela for menor ou igual ao valor especificado.
min-width Aplica estilos se a largura da tela for maior ou igual ao valor especificado.
max-height Aplica estilos se a altura da tela for menor ou igual ao valor especificado.
min-height Aplica estilos se a altura da tela for maior ou igual ao valor especificado.
orientation: portrait Aplica estilos se a tela estiver no modo vertical.
orientation: landscape Aplica estilos se a tela estiver no modo horizontal.

Criando Layout Responsivo

Ajustando o layout conforme a tela

.container { width: 80%; margin: auto; }

@media (max-width: 768px) {
.container {
width: 100%;
padding: 10px;
}
}

Explicação:

  • Para telas acima de 768px, o container tem 80% da largura.
  • ara telas menores que 768px, ele ocupa 100% da largura e ganha um padding de 10px.

Breakpoints Comuns

Mobile First (Comece pelo menor e aumente):

/* Para celulares pequenos */
@media (max-width: 480px) { ... }

/* Para tablets */
@media (max-width: 768px) { ... }

/* Para laptops */
@media (max-width: 1024px) { ... }

/* Para desktops grandes */
@media (max-width: 1200px) { ... }

Desktop First (Comece pelo maior e diminua):

/* Para desktops grandes */
@media (min-width: 1200px) { ... }

/* Para laptops */
@media (min-width: 1024px) { ... }

/* Para tablets */
@media (min-width: 768px) { ... }

/* Para celulares pequenos */
@media (min-width: 480px) { ... }

Media Queries Múltiplas

Podemos combinar várias condições com and, or, e not.

Exemplo: Aplicar estilos em telas entre 600px e 900px

@media (min-width: 600px) and (max-width: 900px) {
body {
background-color: yellow;
}
}

Exemplo: Aplicar estilos APENAS se não for uma tela de impressão

@media not print {
body {
font-size: 16px;
}
}

Media Queries para Temas Escuros

Podemos usar prefers-color-scheme para adaptar o site ao modo escuro.

@media (prefers-color-scheme: dark) {
body {
background-color: black;
color: white;
}
}

Se o usuário ativar o modo escuro no sistema operacional, o site aplicará automaticamente esses estilos.
