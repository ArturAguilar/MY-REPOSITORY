Flexbox

O Flexbox é um sistema de layout no CSS que facilita o alinhamento e a distribuição dos elementos dentro de um contêiner, mesmo quando seus tamanhos são dinâmicos.

Flex Container (Contêiner Flexível)

O Flexbox começa ao definir um elemento como contêiner flexível usando display: flex; ou display: inline-flex;.

Exemplo:

.container {
display: flex; /* Ativa o flexbox */
background-color: #ddd;
padding: 10px;
}

.item {
background-color: #d70000;
color: white;
padding: 20px;
margin: 5px;
}

<div class="container">
<div class="item">1</div>
<div class="item">2</div>
<div class="item">3</div>
</div>

Explicação:

  • display: flex; faz com que os itens internos sejam alinhados em linha por padrão.
  • Os elementos .item agora são flexíveis dentro do .container.

Propriedades do Contêiner Flexível

Propriedade Descrição
display: flex; Ativa o Flexbox no elemento.
flex-direction Define a direção dos itens (row, column, row-reverse, column-reverse).
justify-content Alinha os itens no eixo principal (flex-start, center, flex-end, space-between, space-around, space-evenly).
align-items Alinha os itens no eixo transversal (stretch, flex-start, center, flex-end, baseline).
flex-wrap Permite que os itens quebrem linha (nowrap, wrap, wrap-reverse).
align-content Alinha múltiplas linhas de itens (stretch, center, flex-start, flex-end, space-between, space-around).

Direção dos Itens (flex-direction)

Valor Descrição
row Padrão. Itens alinhados da esquerda para a direita.
row-reverse Itens alinhados da direita para a esquerda.
column Itens alinhados de cima para baixo.
column-reverse Itens alinhados de baixo para cima.

Exemplo - flex-direction: column;

.container {
display: flex;
flex-direction: column;
}

Isso faz com que os itens sejam organizados verticalmente.

Distribuição dos Itens no Eixo Principal (justify-content)

Valor Descrição
flex-start Alinha os itens no início do contêiner.
flex-end Alinha os itens no final do contêiner.
center Centraliza os itens.
space-between Distribui os itens igualmente, com espaço entre eles.
space-around Distribui os itens igualmente, com espaço ao redor deles.
space-evenly Espaço igual entre os itens e nas bordas do contêiner.

Exemplo - justify-content: space-between;

.container {
display: flex;
justify-content: space-between;
}

Os itens ficam alinhados com espaço igual entre eles.

Alinhamento no Eixo Transversal (align-items)

Valor Descrição
stretch Padrão. Os itens ocupam toda a altura disponível.
flex-start Alinha os itens no topo do contêiner.
flex-end Alinha os itens na parte inferior.
center Centraliza os itens verticalmente.
baseline Alinha os itens com base no texto dentro deles.

Exemplo - align-items: center;

.container {
display: flex;
align-items: center;
}

Os itens ficarão centralizados verticalmente.

Controle de Tamanho dos Itens (flex-grow, flex-shrink, flex-basis)

Propriedade Descrição
flex-grow Define quanto um item pode crescer comparado aos outros.
flex-shrink Define quanto um item pode encolher quando não há espaço suficiente.
flex-basis Define o tamanho inicial do item antes da distribuição do espaço.

Exemplo - flex-grow

.item:first-child {
flex-grow: 2;
}

.item:last-child {
flex-grow: 1;
}

O primeiro item ocupará o dobro de espaço comparado ao último.

Flex-Wrap: Quebra de Linha Automática

Valor Descrição
nowrap Padrão. Todos os itens ficam na mesma linha.
wrap Itens quebram linha quando não há espaço suficiente.
wrap-reverse Quebra de linha invertida.

Exemplo - flex-wrap: wrap;

.container {
display: flex;
flex-wrap: wrap;
}

Os itens quebram linha automaticamente quando necessário.

Flexbox Responsivo

Podemos usar Media Queries para adaptar o layout em telas menores.

Exemplo de Layout Responsivo

.container {
display: flex;
flex-wrap: wrap;
justify-content: center;
}

@media (max-width: 600px) {
.container {
flex-direction: column;
}
}

  • Acima de 600px: os itens ficam em linha.
  • Abaixo de 600px: os itens ficam em coluna.
