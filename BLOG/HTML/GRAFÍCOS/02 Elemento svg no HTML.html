Elemento svg no HTML

O SVG (Scalable Vector Graphics) é um formato baseado em XML para criar gráficos vetoriais diretamente no HTML.

✅ Vantagens do SVG:

  • Escalável sem perder qualidade.
  • Personalizável via CSS e JavaScript.
  • Menos peso para gráficos simples.

Criando um SVG Simples

Podemos desenhar um círculo usando svg e circle.

<svg width="200" height="200">
<circle cx="100" cy="100" r="50" stroke="black" stroke-width="2" fill="red" />
</svg>

✅ Desenha um círculo vermelho de raio 50px no centro (100,100).

Desenhando Formas com SVG

Retângulo (<rect>)

<svg width="200" height="200">
<rect x="20" y="20" width="100" height="50" fill="blue" stroke="black" stroke-width="2"/>
</svg>

✅ Um retângulo azul de 100x50px na posição (20,20).

Linha (<line>)

<svg width="200" height="200">
<line x1="10" y1="10" x2="150" y2="100" stroke="black" stroke-width="2"/>
</svg>

✅ Uma linha preta de (10,10) a (150,100).

Polígono (<polygon>)

<svg width="200" height="200">
<polygon points="50,10 90,90 10,90" fill="green" stroke="black" stroke-width="2"/>
</svg>

✅ Um triângulo verde.

Texto no SVG (<text>)

<svg width="200" height="100">
<text x="10" y="40" font-size="20" fill="black">Olá, SVG!</text>
</svg>

✅ Exibe "Olá, SVG!" em preto.

Criando Gradientes

Podemos adicionar efeitos de gradiente linear.

<svg width="200" height="200">
<defs>
<linearGradient id="gradiente" x1="0%" y1="0%" x2="100%" y2="0%">
<stop offset="0%" stop-color="blue"/>
<stop offset="100%" stop-color="red"/>
</linearGradient>
</defs>
<rect x="10" y="10" width="180" height="100" fill="url(#gradiente)"/>
</svg>

✅ Um retângulo com gradiente azul → vermelho.

Animando um SVG

Podemos animar elementos sem JavaScript usando animate.

<svg width="200" height="200">
<circle cx="50" cy="50" r="20" fill="blue">
<animate attributeName="cx" from="50" to="150" dur="2s" repeatCount="indefinite"/>
</circle>
</svg>

✅ O círculo azul se move de (50,50) → (150,50) continuamente.
