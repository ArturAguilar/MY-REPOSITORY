Tudo sobre tabelas em HTML

As tabelas em HTML são criadas com a tag <table> e são usadas para exibir dados de forma tabular, com linhas e colunas.

Estrutura Básica de uma Tabela

  • <table>: Define a tabela.
  • <tr>: Define uma linha da tabela (table row).
  • <th>: Define uma célula de cabeçalho (table header). Os textos dentro da tag <th> geralmente são exibidos em negrito e centralizados.
  • <td>: Define uma célula de dados (table data), ou seja, os valores dentro das linhas da tabela.

Exemplo Básico

<table>
<tr>
<th>Nome</th>
<th>Idade</th>
</tr>
<tr>
<td>Artur</td>
<td>18</td>
</tr>
<tr>
<td>João</td>
<td>25</td>
</tr>
</table>

Atributos de Tabelas


border: Define a largura da borda da tabela.

<table border="1">

cellpadding: Define o espaço entre as células da tabela e seu conteúdo.

<table cellpadding="10">

cellspacing: Define o espaço entre as células.

<table cellspacing="5">

Agrupamento de Células


<thead>: Define o cabeçalho da tabela.

Contém as células de cabeçalho, geralmente com <th>.

<thead>
<tr>
<th>Nome</th>
<th>Idade</th>
</tr>
</thead>

<tbody>: Define o corpo da tabela.

Contém as linhas de dados da tabela.

<tbody>
<tr>
<td>Artur</td>
<td>18</td>
</tr>
</tbody>

<tfoot>: Define o rodapé da tabela.

Pode conter informações de resumo ou totalizações.

<tfoot>
<tr>
<td>Total</td>
<td>43</td>
</tr>
</tfoot>

Exemplo Completo de Tabela

<!DOCTYPE html>
<html>
<head>
<title>Exemplo de Tabela</title>
</head>
<body>

<h1>Tabela de Idades</h1>

<table border="1" cellpadding="10">
<thead>
<tr>
<th>Nome</th>
<th>Idade</th>
</tr>
</thead>
<tbody>
<tr>
<td>Artur</td>
<td>18</td>
</tr>
<tr>
<td>João</td>
<td>25</td>
</tr> </tbody>
<tfoot>
<tr>
<td>Total</td>
<td>43</td>
</tr>
</tfoot>
</table>

</body>
</html>

Bordas em Tabelas HTML

Por padrão, as tabelas HTML não possuem bordas. É possível adicionar bordas às tabelas usando CSS ou o atributo border diretamente no elemento <table>.

Exemplo de Tabela com Bordas Usando o Atributo border

<table border="1">
<tr>
<th>Nome</th>
<th>Idade</th>
</tr>
<tr>
<td>Artur</td>
<td>18</td>
</tr>
</table>

O atributo border="1" adiciona uma borda simples à tabela.

Bordas Estilizadas com CSS

Adicionando Bordas à Tabela

Use a propriedade border no CSS para estilizar as bordas.

<style>
table, th, td {
border: 1px solid black;
}
</style>

Tabela com CSS:

<table>
<tr>
<th>Nome</th>
<th>Idade</th>
</tr>
<tr>
<td>Artur</td>
<td>18</td>
</tr>
</table>

Controlando Bordas Colapsadas

border-collapse

Controla se as bordas da tabela e das células serão separadas ou colapsadas.

Exemplo com border-collapse: collapse;:

<style>
table {
border-collapse: collapse;
}
th, td {
border: 1px solid black;
}
</style>


Resultado: As bordas das células se juntam em uma única borda.

border-spacing

Define o espaço entre as bordas das células (se border-collapse estiver configurado como separate).

<style>
table {
border-collapse: separate;
border-spacing: 10px;
}
</style>

Exemplo Completo

<!DOCTYPE html>
<html>
<head>
<style>
table {
width: 50%;
border-collapse: collapse;
}
th, td {
border: 1px solid black;
padding: 10px;
text-align: left;
}
</style>
</head>
<body>

<h1>Tabela com Bordas Estilizadas</h1>

<table>
<tr>
<th>Nome</th>
<th>Idade</th>
</tr>
<tr>
<td>Artur</td>
<td>18</td>
</tr>
<tr>
<td>João</td>
<td>25</td>
</tr>
</table>

</body>
</html>

Tamanho de Tabelas em HTML

O tamanho das tabelas em HTML pode ser controlado com os atributos width e height (considerados obsoletos) ou, preferencialmente, com CSS. Isso permite ajustar a largura e a altura da tabela ou de células específicas.

Controlando o Tamanho com o Atributo width

O atributo width define a largura da tabela ou célula em pixels ou porcentagem.

Exemplo:

<table width="50%" border="1">
<tr>
<th>Nome</th>
<th>Idade</th>
</tr>
<tr>
<td>Artur</td>
<td>18</td>
</tr>
</table>

Controlando o Tamanho com CSS

A maneira recomendada para controlar o tamanho das tabelas é utilizando CSS. Isso oferece mais flexibilidade e controle.

Definir Largura e Altura da Tabela

<style>
table {
width: 100%;
height: 200px;
border: 1px solid black;
}
th, td {
border: 1px solid black;
}
</style>

Exemplo:

<table>
<tr>
<th>Nome</th>
<th>Idade</th>
</tr>
<tr>
<td>Artur</td>
<td>18</td>
</tr>
</table>

Controlando o Tamanho de Células Específicas

Você pode definir o tamanho de células específicas com o uso de CSS no elemento <th> ou <td>.

Exemplo:

<style>
th, td {
width: 150px;
height: 50px;
text-align: center;
}
</style>

<table border="1">
<tr>
<th>Nome</th>
<th>Idade</th>
</tr>
<tr>
<td>Artur</td>
<td>18</td>
</tr>
</table>

Mesclando Largura e Altura

Para tabelas dinâmicas, defina a largura em porcentagem para que o layout seja responsivo.

Exemplo Responsivo:

<style>
table {
width: 80%;
height: auto;
border: 1px solid black;
}
th, td {
padding: 10px;
text-align: left;
} </style>

<table>
<tr>
<th>Nome</th>
<th>Idade</th>
</tr>
<tr>
<td>Artur</td>
<td>18</td>
</tr>
<tr>
<td>João</td>
<td>25</td>
</tr>
</table>

Cabeçalhos em Tabelas HTML

Os cabeçalhos das tabelas em HTML são definidos com o elemento <th> (table header). Eles são usados para destacar o conteúdo das colunas ou linhas, normalmente exibindo o texto em negrito e centralizado.

Estrutura Básica de uma Tabela com Cabeçalhos

  • O elemento <th> é usado para definir células de cabeçalho.
  • Ele pode ser colocado dentro de um <tr> (linha da tabela).

Exemplo:

<table border="1">
<tr>
<th>Nome</th>
<th>Idade</th>
<th>Cidade</th>
</tr>
<tr>
<td>Artur</td>
<td>18</td>
<td>São João de Meriti</td>
</tr>
<tr>
<td>João</td>
<td>25</td>
<td>Rio de Janeiro</td>
</tr>
</table>

Posicionamento e Funções dos Cabeçalhos

Cabeçalhos de Colunas

Normalmente aparecem na primeira linha da tabela e identificam o conteúdo de cada coluna.

Cabeçalhos de Linhas

Podem ser usados na primeira célula de cada linha para identificar o conteúdo da linha.

Exemplo de Cabeçalhos de Linhas:

<table border="1">
<tr>
<th></th>
<th>Produto 1</th>
<th>Produto 2</th>
</tr>
<tr>
<th>Preço</th>
<td>$10</td>
<td>$15</td>
</tr>
</table>

Atributo scope no Elemento <th>

O atributo scope é usado para melhorar a acessibilidade, ajudando tecnologias assistivas (como leitores de tela) a entender a relação entre cabeçalhos e células.

  • scope="col": Indica que o cabeçalho se aplica à coluna.
  • scope="row": Indica que o cabeçalho se aplica à linha.
  • scope="colgroup" ou scope="rowgroup": Indicam grupos de colunas ou linhas.

Exemplo com scope:

<table border="1">
<tr>
<th scope="col">Nome</th>
<th scope="col">Idade</th>
<th scope="col">Cidade</th>
</tr>
<tr>
<td>Artur</td>
<td>18</td>
<td>São João de Meriti</td>
</tr>
</table>

Estilizando Cabeçalhos com CSS

Você pode usar CSS para personalizar os cabeçalhos da tabela, como cores de fundo, alinhamento, ou bordas.

Exemplo de Estilo para <th>:

<style>
th {
background-color: #d70000; /* Vermelho */
color: white;
text-align: left;
padding: 10px;
}
table {
border-collapse: collapse;
width: 100%;
}
th, td {
border: 1px solid black;
}
</style>

Tabela com Estilização:

<table>
<tr>
<th>Nome</th>
<th>Idade</th>
<th>Cidade</th>
</tr>
<tr>
<td>Artur</td>
<td>18</td>
<td>São João de Meriti</td>
</tr>
</table>

Espaçamento Interno e Externo em Tabelas

Espaçamento Interno (padding):

Define o espaço dentro das células da tabela, entre o conteúdo e as bordas das células.

Espaçamento Externo (border-spacing):

Define o espaço entre as células da tabela.

Controlando o Espaçamento Interno (Padding)

Você pode ajustar o espaçamento interno das células usando a propriedade padding no CSS.

Exemplo:

<style>
td, th {
padding: 15px; /* Espaçamento interno de 15px */
}
</style>

<table border="1">
<tr>
<th>Nome</th>
<th>Idade</th>
<th>Cidade</th>
</tr>
<tr>
<td>Artur</td>
<td>18</td>
<td>São João de Meriti</td>
</tr>
</table>

Controlando o Espaçamento Externo (Border-Spacing)

A propriedade border-spacing ajusta o espaço entre as células.

Exemplo:

<style>
table {
border-spacing: 10px; /* Espaço entre as células */
}
</style>

<table border="1">
<tr>
<th>Produto</th>
<th>Preço</th>
</tr>
<tr>
<td>Camiseta</td>
<td>R$ 50</td>
</tr>
</table>

Removendo o Espaçamento Entre Células (border-collapse)

A propriedade border-collapse permite remover o espaçamento entre as células, fazendo com que suas bordas se fundam.

  • border-collapse: collapse;: Remove o espaçamento.
  • border-collapse: separate; (valor padrão): Mantém o espaçamento entre as bordas das células.

Exemplo com border-collapse:

<style>
table {
border-collapse: collapse;
}
td, th {
border: 1px solid black;
padding: 10px;
}
</style>
<table>
<tr>
<th>Nome</th>
<th>Idade</th>
</tr>
<tr>
<td>Artur</td>
<td>18</td>
</tr>
</table>

Exemplo Completo: Ajustando Padding e Border-Spacing

<style>
table {
border-spacing: 20px; /* Espaço entre células */
border-collapse: separate; /* Mantém o espaçamento
externo */
}
td, th {
padding: 15px; /* Espaçamento interno */
border: 1px solid black;
}
</style>

<table>
<tr>
<th>Produto</th>
<th>Preço</th>
</tr>
<tr>
<td>Notebook</td>
<td>R$ 3.000</td>
</tr>
<tr>
<td>Smartphone</td>
<td>R$ 1.200</td>
</tr>
</table>

Mesclando Células em Tabelas com colspan e rowspan

colspan:

  • Define o número de colunas que uma célula deve ocupar.
  • É usado quando você deseja mesclar células horizontalmente.

Sintaxe:

<td colspan="n">Conteúdo</td>

Exemplo com colspan:

<table border="1">
<tr>
<th>Nome</th>
<th>Idade</th>
<th>Cidade</th>
</tr>
<tr>
<td colspan="2">Artur</td>
<td>São João de Meriti</td>
</tr>
</table>

rowspan:

  • Define o número de linhas que uma célula deve ocupar.
  • É usado para mesclar células verticalmente.

Sintaxe:

<td rowspan="n">Conteúdo</td>

Exemplo com rowspan:

<table border="1">
<tr>
<th>Produto</th>
<th>Quantidade</th>
<th>Preço</th>
</tr>
<tr>
<td rowspan="2">Camiseta</td>
<td>1</td>
<td>R$ 50</td>
</tr>
<tr>
<td>2</td>
<td>R$ 100</td>
</tr>
</table>

Combinando colspan e rowspan

Você pode usar colspan e rowspan juntos para criar layouts mais complexos em tabelas.

Exemplo:

<table border="1">
<tr>
<th>Nome</th>
<th colspan="2">Detalhes</th>
</tr>
<tr>
<td rowspan="2">Artur</td>
<td>Idade</td>
<td>18</td>
</tr>
<tr>
<td>Cidade</td>
<td>São João de Meriti</td>
</tr>
</table>

Tabelas podem ser estilizadas usando CSS para melhorar sua aparência e torná-las mais legíveis. Aqui estão os elementos principais:

Bordas em Tabelas

Use a propriedade border para adicionar bordas às tabelas, linhas e células.

Exemplo:

}
</style>

<th>Idade</th>
</tr>

<td>18</td>
</tr>

Use border-collapse para colapsar bordas das células, criando uma borda única entre elas.

<style>

<tr>
<th>Produto</th>
<th>Preço</th>
</tr>

<td>Camiseta</td>
<td>R$ 50</td>
</tr>

Adicione espaço dentro das células usando padding.

Exemplo:

<style>

<table border="1">
<tr>
<th>Nome</th>
<th>Idade</th>
</tr>
<tr>
<td>Artur</td>
<td>18</td>
</tr>

Alinhamento de Texto

Use text-align para alinhar o texto nas células.

Exemplo:

<style>

</style>

Use width e height para definir as dimensões da tabela ou das células.

Exemplo:

Adicione cores de fundo ou texto usando background-color e color.

<style>

<td>18</td>
</tr>

O Elemento <colgroup>

  • O elemento <colgroup> é usado para agrupar colunas em uma tabela e aplicar estilo ou atributos às colunas de forma coletiva.
  • Ele deve ser usado dentro da tabela e antes do conteúdo do corpo da tabela (antes das linhas <tr>).

Atributos do <colgroup>

  • span: Define o número de colunas que o grupo cobre.
  • Se omitido, cada <col> dentro do <colgroup> representa uma única coluna.

O Elemento <col>

Dentro do <colgroup>, usamos o elemento <col> para especificar configurações ou estilos para colunas individuais ou grupos de colunas.

Exemplo Básico de <colgroup>

Este exemplo aplica estilos diferentes às primeiras duas colunas da tabela:

<style>
.col1 {
background-color: #d70000; /* Vermelho */
color: white;
}
.col2 {
background-color: #f5f5f5; /* Cinza claro */
}
</style>

<table border="1">
<colgroup>
<col class="col1">
<col class="col2">
</colgroup>

<tr>
<th>Nome</th>
<th>Idade</th>
</tr>
<tr>
<td>Artur</td>
<td>18</td>
</tr>
<tr>
<td>Maria</td>
<td>22</td>
</tr>
</table>

Usando o Atributo span

Se várias colunas devem ter o mesmo estilo, você pode usar span para agrupá-las:

<style>
.highlight {
background-color: #d70000;
color: white;
}
</style>

<table border="1">
<colgroup>
<col span="2" class="highlight">
</colgroup>

<tr>
<th>Nome</th>
<th>Idade</th>
<th>Cidade</th>
</tr>
<tr>
<td>Artur</td>
<td>18</td>
<td>Rio de Janeiro</td>
</tr>
</table>

Exemplo:

<style>
.wide {
width: 200px;
}
</style>

<table border="1">
<colgroup>
<col class="wide">
<col>
</colgroup>

<tr>
<th>Nome</th>
<th>Idade</th>
</tr>
<tr>

<td>18</td>
