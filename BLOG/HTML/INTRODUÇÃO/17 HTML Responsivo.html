O Que é HTML Responsivo?

  • Definição: HTML responsivo é a técnica de criar páginas da web que se ajustam automaticamente ao tamanho e resolução da tela do dispositivo (desktop, tablet ou smartphone).
  • Objetivo: Melhorar a experiência do usuário em diferentes dispositivos, garantindo que o conteúdo seja legível e fácil de navegar.

Elementos-Chave do HTML Responsivo

Meta Viewport

  • Essencial para páginas responsivas, pois controla a forma como o conteúdo é exibido em dispositivos móveis.
  • Atributos:
    • width=device-width: Faz a largura da página corresponder à largura da tela do dispositivo.
    • initial-scale=1.0: Define o zoom inicial.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

CSS Media Queries

  • Adapta o layout da página com base no tamanho da tela.
  • Como funciona:
    • O exemplo acima altera o fundo da página para azul claro quando a largura da tela é menor ou igual a 600px.

@media (max-width: 600px) {
body {
background-color: lightblue;
}
}

Larguras Máximas Relativas (max-width)

  • Usar valores relativos como % ou em em vez de valores fixos como px.

img {
max-width: 100%;
height: auto;
}

Grade Flexível (Flexbox e Grid)

  • Permite criar layouts que se ajustam automaticamente.

Exemplo com Flexbox:

.container {
display: flex;
flex-wrap: wrap;
}
.item {
flex: 1 1 auto;
}

Exemplo com Grid:

.container {
display: grid;
grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
}

Imagens Responsivas

  • As imagens também precisam se ajustar ao tamanho da tela:

img {
max-width: 100%;
height: auto;
}

Usando o picture

  • Para carregar imagens diferentes com base na resolução da tela.

<picture>
<source srcset="imagem-grande.jpg" media="(min-width: 800px)">
<source srcset="imagem-pequena.jpg" media="(max-width: 799px)">
<img src="imagem-padrao.jpg" alt="Imagem Responsiva">
</picture>

Exemplo Completo de Página Responsiva

<!DOCTYPE html>
<html lang="pt-br">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Página Responsiva</title>
<style>
body {
font-family: Arial, sans-serif;
margin: 0;
}

header {
background-color: #d70000;
color: white;
text-align: center;
padding: 10px;
}

.container {
display: grid;
grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
gap: 10px;
padding: 10px;
}

.item {
background-color: #f4f4f4;
border: 1px solid #ccc;
text-align: center;
padding: 20px;
}
</style>
</head>
<body>
<header>Cabeçalho Responsivo</header>

<div class="container">
<div class="item">Item 1</div>
<div class="item">Item 2</div>
<div class="item">Item 3</div>
<div class="item">Item 4</div>

</div>
</body>
</html>
