Conversão de Tipos de Dados

O JavaScript é uma linguagem dinamicamente tipada, o que significa que o tipo de uma variável é determinado automaticamente no momento da atribuição e pode mudar ao longo do tempo. Para entender melhor como lidar com os tipos de dados e convertê-los corretamente, vamos explorar o operador typeof e os métodos de conversão de tipos.

O Operador typeof

O operador typeof é usado para verificar o tipo de um valor ou variável em JavaScript. Ele retorna uma string com o nome do tipo do dado.

Sintaxe

typeof valor;
typeof (valor);

Exemplo

console.log(typeof "Hello"); // "string"
console.log(typeof 42); // "number"
console.log(typeof true); // "boolean"
console.log(typeof undefined); // "undefined"
console.log(typeof null); // "object" (BUG histórico do JS)
console.log(typeof {}); // "object"
console.log(typeof []); // "object" (arrays são objetos)
console.log(typeof function(){});// "function"
console.log(typeof Symbol()); // "symbol"
console.log(typeof BigInt(10)); // "bigint"

⚠ Atenção: typeof null retorna "object", o que é um erro da linguagem que nunca foi corrigido.

Conversão de Tipos (Type Conversion)

Às vezes, é necessário converter valores de um tipo para outro. O JavaScript permite conversão implícita (feita automaticamente) e explícita (feita manualmente com funções específicas).

Conversão para String (String())

Podemos converter valores para string usando String(valor) ou toString().
