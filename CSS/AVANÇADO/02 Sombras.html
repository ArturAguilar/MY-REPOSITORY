Sombras

O CSS3 permite adicionar sombras a textos e elementos para criar efeitos visuais elegantes. Existem dois tipos principais de sombras:

  • Sombra de texto (text-shadow) – Aplica sombra a textos.
  • Sombra de caixa (box-shadow) – Aplica sombra a elementos (como div, button, etc.).

Sombra de Texto (text-shadow)

A propriedade text-shadow permite criar efeitos de profundidade e brilho nos textos.

Sintaxe:

text-shadow: deslocamentoX deslocamentoY desfoque cor;

Exemplo Simples:

h1 {
text-shadow: 2px 2px 5px gray;
}

Explicação dos Valores:

Propriedade Descrição
deslocamentoX Distância horizontal da sombra (positivo para direita, negativo para esquerda)
deslocamentoY Distância vertical da sombra (positivo para baixo, negativo para cima)
desfoque Intensidade do desfoque (opcional)
cor Cor da sombra

Exemplo com Multicamadas:

h1 {
text-shadow: 2px 2px 5px red, -2px -2px 5px blue;
}

Exemplo de Efeito Neon:

h1 {
color: white;
text-shadow: 0 0 10px cyan, 0 0 20px blue, 0 0 30px cyan;
}

Sombra de Caixa (box-shadow)

A propriedade box-shadow adiciona sombra em elementos como div, button, input, etc.

Sintaxe:

box-shadow: deslocamentoX deslocamentoY desfoque espalhamento cor;

Exemplo Simples:

div {
width: 200px;
height: 100px;
background: lightblue;
box-shadow: 5px 5px 10px gray;
}

Explicação dos Valores:

Propriedade Descrição
deslocamentoX Distância horizontal da sombra (positivo para direita, negativo para esquerda)
deslocamentoY Distância vertical da sombra (positivo para baixo, negativo para cima)
desfoque Intensidade do desfoque (opcional)
espalhamento Aumenta ou reduz a área da sombra (opcional)
cor Cor da sombra

Exemplo com Espalhamento:

div {
box-shadow: 5px 5px 10px 5px rgba(0, 0, 0, 0.5);
}

Exemplo de Sombra Interna (inset):

div {
box-shadow: inset 5px 5px 10px rgba(0, 0, 0, 0.5);
}

Exemplo de Sombra com Várias Camadas:

div {
box-shadow: 5px 5px 10px gray, -5px -5px 10px lightgray;
}

Exemplo Completo com text-shadow e box-shadow

body {
background: #f0f0f0;
}

h1 {
color: #d70000;
text-shadow: 3px 3px 5px black, -3px -3px 5px gray;
}

div {
width: 200px;
height: 100px;
background: white;
margin: 50px auto;
box-shadow: 10px 10px 20px rgba(0, 0, 0, 0.5);
}
