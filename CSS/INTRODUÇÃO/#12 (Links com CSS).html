Estilizando Links com CSS

No CSS, os links podem ser estilizados para melhorar a aparência e a usabilidade de uma página web. É possível aplicar diversas propriedades de estilo, como cor, fonte, fundo, entre outras, para personalizar os links de acordo com o design desejado.

Exemplo Básico:

a {
color: hotpink;
}

Estados dos Links

Os links possuem diferentes estados que podem ser configurados individualmente para fornecer feedback visual ao usuário. Os quatro principais estados são:

  • a:link: Um link normal, não visitado.
  • a:visited: Um link que o usuário já visitou.
  • a:hover: Um link quando o usuário passa o mouse sobre ele.
  • a:active: Um link no momento em que é clicado.

Exemplo:

/* Link não visitado */
a:link {
color: red;
}

/* Link visitado */
a:visited {
color: green;
}

/* Link ao passar o mouse */
a:hover {
color: hotpink;
}

/* Link ativo */
a:active {
color: blue;
}

Neste exemplo, os links terão núcleos diferentes dependendo do seu estado:

  • Não visitado: vermelho.
  • Visitado: verde.
  • Ao passar o mouse: rosa.
  • Ativo (no momento do clique): azul.

Importante: Ao definir estilos para os diferentes estados dos links, a ordem das pseudoclasses é crucial:

  • a:hoverdeve vir após a:linke a:visited.
  • a:activedeve vir após a:hover.

Removendo Sublinhado dos Links

Por padrão, os links são sublinhados. Para remover o sublinhado, utilize uma propriedade text-decoration:

Exemplo:

a:link, a:visited {
text-decoration: none;
}

a:hover, a:active {
text-decoration: underline;
}

Alterando o Cor de Fundo dos Links

Você pode alterar o fundo dos links em diferentes estados para fornecer feedback visual adicional:

Exemplo:

a:link {
background-color: yellow;
}

a:visited {
background-color: cyan;
}

a:hover {
background-color: lightgreen;
}

a:active {
background-color: hotpink;
}

Estilizando Links como Botões

Para estilizar links de forma que se parecem e se comportam como botões, você pode combinar várias propriedades CSS:

Exemplo:

a:link, a:visited {
background-color: #f44336;
color: white;
padding: 14px 25px;
text-align: center;
text-decoration: none;
display: inline-block;
}

a:hover, a:active {
background-color: red;
}

E no HTML:

<a href="https://www.exemplo.com"vEste é um link</a>
