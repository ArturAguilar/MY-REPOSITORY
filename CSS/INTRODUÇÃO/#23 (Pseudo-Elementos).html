Pseudo-Elementos

Os pseudo-elementos em CSS são usados para aplicar estilos a partes específicas de um elemento. Eles permitem estilizar elementos que não podem ser selecionados diretamente no HTML, como a primeira letra de um parágrafo ou o conteúdo gerado automaticamente antes ou depois de um elemento.

Pseudo-Elemento ::before

Insere conteúdo antes do conteúdo de um elemento.

div::before {
content: "Nota: ";
font-weight: bold;
}

Explicação:

  • O texto "Nota: " será adicionado antes do conteúdo de cada div, e o texto será em negrito.
  • O content é obrigatório quando se usa ::before ou ::after.

Pseudo-Elemento ::after

Insere conteúdo após o conteúdo de um elemento.

div::after {
content: " (finalizado)";
color: green;
}

Explicação:

  • O texto "(finalizado)" será adicionado após o conteúdo de cada div, e a cor será verde.

Pseudo-Elemento ::first-letter

Seleciona a primeira letra de um bloco de texto e aplica um estilo específico.

p::first-letter {
font-size: 2em;
color: red;
}

Explicação:

A primeira letra de cada parágrafo (p) será aumentada e colorida de vermelho.

Pseudo-Elemento ::first-line

Seleciona a primeira linha de um bloco de texto e aplica um estilo específico.

p::first-line {
font-weight: bold;
}

Explicação:

  • A primeira linha de cada parágrafo será em negrito.
  • Esse pseudo-elemento é aplicável a blocos de texto e considera a largura da área do contêiner.
