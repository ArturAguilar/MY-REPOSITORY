Forms

Os formulários são essenciais para interações em sites, e o CSS nos permite personalizá-los para melhorar a experiência do usuário. Podemos estilizar inputs, labels, botões e até mensagens de erro.

Estilizando Inputs e Labels

Os principais tipos de elementos de formulário incluem:

  • ✅ input (text, password, email, number, etc.)
  • ✅ textarea
  • ✅ select
  • ✅ button
  • ✅ label

Podemos personalizar esses elementos para deixá-los mais atraentes.

/* Estilizando o rótulo */
label {
font-size: 16px;
font-weight: bold;
color: #333;
margin-bottom: 5px;
display: block;
}

/* Estilizando campos de entrada */
input[type="text"],
input[type="email"],
input[type="password"],
textarea {
width: 100%;
padding: 10px;
margin: 5px 0;
border: 2px solid #d70000;
border-radius: 5px;
font-size: 16px;
}

/* Estilizando os campos de entrada ao focar */
input:focus,
textarea:focus {
border-color: #ff4500;
outline: none;
background-color: #f9f9f9;
}

/* Botões estilizados */
button {
background-color: #d70000;
color: white;
border: none;
padding: 10px 20px;
cursor: pointer;
border-radius: 5px;
font-size: 16px;
}

/* Efeito de hover no botão */
button:hover {
background-color: #a50000;
}

Estilizando select e checkbox

Os select e checkbox também podem ser estilizados.

/* Estilizando o campo de seleção */
select {
width: 100%;
padding: 10px;
border: 2px solid #d70000;
border-radius: 5px;
font-size: 16px;
}


/* Estilizando checkboxes e radio buttons */
input[type="checkbox"],
input[type="radio"] {
accent-color: #d70000; /* Cor personalizada */
width: 18px;
height: 18px;
}

<label for="gender">Gênero:</label>
<select id="gender">
<option value="male">Masculino</option>
<option value="female">Feminino</option>
<option value="other">Outro</option>
</select>
