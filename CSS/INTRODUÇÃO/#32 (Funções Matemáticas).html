Funções Matemáticas

As funções matemáticas no CSS permitem realizar cálculos diretamente dentro das regras CSS, tornando o design mais flexível e dinâmico.

Principais Funções Matemáticas

calc() – Realiza Cálculos

A função calc() permite operações matemáticas simples para definir valores no CSS.

.container {
width: calc(100% - 50px); /* Reduz 50px da largura total */
height: calc(50vh + 20px); /* 50% da altura da tela + 20px */
}

  • Permite adição (+), subtração (-), multiplicação (*) e divisão (/).
  • Pode combinar unidades diferentes (px, %, em, rem, vw, vh).

min() – Retorna o Menor Valor

Define um valor baseado no menor dos valores informados.

.box {
width: min(50vw, 300px); /* Usa 50% da largura da tela ou 300px, o que for menor */
}

Útil para tornar elementos responsivos sem ultrapassar um limite máximo.

max() – Retorna o Maior Valor

Define um valor baseado no maior dos valores informados.

.box {
width: max(200px, 50%); /* Usa 200px ou 50% da largura, o que for maior */
}

Garante que um elemento nunca fique pequeno demais.

clamp() – Define Mínimo, Preferido e Máximo

A função clamp() define um valor que se ajusta automaticamente, dentro de um intervalo mínimo e máximo.

.texto {
font-size: clamp(16px, 2vw, 24px);
}

  • clamp(min, preferido, max)
  • Garante acessibilidade: o tamanho do texto varia, mas nunca fica pequeno demais nem grande demais.

Exemplo Prático

.container {
width: calc(100% - 50px);
height: min(80vh, 600px);
padding: clamp(10px, 2vw, 30px);
}

  • Flexibilidade para layouts responsivos.
  • Evita media queries desnecessárias.
