🖌️ Bordas no CSS

O CSS permite personalizar bordas de elementos com diversas propriedades.

✏️ border (Shorthand)

A propriedade border permite definir largura, estilo e cor em uma única linha.

p {
border: 2px solid red;
}

Isso equivale a:

p {
border-width: 2px;
border-style: solid;
border-color: red;
}

📏 border-width (Largura da Borda)

Define a espessura da borda.

Opções:

border-width: thin; /* Fina */
border-width: medium; /* Média (padrão) */
border-width: thick; /* Grossa */
border-width: 5px; /* Valor específico */

🔹 Também pode definir diferentes valores para cada lado:

border-width: 2px 5px 10px 3px; /* Topo Direita Base Esquerda */

🎨 border-style (Estilo da Borda)

Define o tipo de linha da borda.

Opções disponíveis:

border-style: solid; /* Linha sólida */
border-style: dashed; /* Tracejada */
border-style: dotted; /* Pontilhada */
border-style: double; /* Linha dupla */
border-style: groove; /* Efeito 3D */
border-style: ridge; /* Efeito elevado */
border-style: inset; /* Aparência rebaixada */
border-style: outset; /* Aparência levantada */
border-style: none; /* Sem borda */

🔹 Exemplo prático:

div {
border: 3px dashed blue;
}

🌈 border-color (Cor da Borda)

Define a cor da borda.

Pode usar:

  • Nomes de cores: red, blue, green
  • Hexadecimal: #ff0000
  • RGB: rgb(255, 0, 0)
  • HSL: hsl(0, 100%, 50%)

🔹 Exemplo prático:

div {
border: 5px solid;
border-color: red green blue yellow; /* Topo Direita Base Esquerda */
}

🟦 border-radius (Arredondamento da Borda)

Define bordas arredondadas.

Exemplo:

div {
border: 2px solid black;
border-radius: 10px;
}

🔹 Também pode definir valores para cada canto:

border-radius: 10px 20px 30px 40px; /* TopoEsquerda, TopoDireita, BaseDireita, BaseEsquerda */

🔹 Para criar um círculo:

border-radius: 50%;

📏 Largura da Borda no CSS

A propriedade border-width define a espessura da borda de um elemento.

🎯 Sintaxe

elemento {
border-width: valor;
}

Onde valor pode ser um tamanho específico ou uma palavra-chave.

🏷️ Valores Possíveis

🔹 Palavras-chave (Valores Padrão)

border-width: thin; /* Fina */
border-width: medium; /* Média (padrão) */
border-width: thick; /* Grossa */

🔹 Valores numéricos

Especifica a largura da borda em px, em, rem, %, etc.

border-width: 2px; /* 2 pixels */
border-width: 0.5em; /* 0.5 vezes o tamanho da fonte */
border-width: 10%; /* 10% do tamanho do contêiner */

🖼️ Diferentes Lados da Borda

É possível definir valores individuais para cada lado:

border-width: 5px 10px 15px 20px;
/* Topo Direita Base Esquerda */

🔹 Alternativamente, usar propriedades separadas:

border-top-width: 5px;
border-right-width: 10px;
border-bottom-width: 15px;
border-left-width: 20px;

🎨 Exemplo Prático

.box {
border-style: solid; /* Necessário definir um estilo */
border-width: 5px; /* Borda de 5px em todos os lados */
border-color: blue; }

Cor da Borda no CSS

A propriedade border-color define a cor da borda de um elemento.

Sintaxe

elemento {
border-color: valor;
}

Onde valor pode ser um nome de cor, um código hexadecimal, um RGB, um HSL ou mesmo transparent.

Valores Possíveis

Cores nomeadas

border-color: red;
border-color: blue;
border-color: green;

Código hexadecimal

border-color: #ff0000; /* Vermelho */
border-color: #00ff00; /* Verde */
border-color: #0000ff; /* Azul */

RGB (Red, Green, Blue)

border-color: rgb(255, 0, 0); /* Vermelho */
border-color: rgb(0, 255, 0); /* Verde */
border-color: rgb(0, 0, 255); /* Azul */
border-color: rgba(255, 0, 0, 0.5); /* Vermelho com 50% de transparência */

HSL (Hue, Saturation, Lightness)

border-color: hsl(0, 100%, 50%); /* Vermelho */
border-color: hsl(120, 100%, 50%); /* Verde */
border-color: hsl(240, 100%, 50%); /* Azul */
border-color: hsla(0, 100%, 50%, 0.5); /* Vermelho com 50% de transparência */

Cor Transparente

border-color: transparent;

Útil para ocultar uma borda sem removê-la completamente.

Diferentes Cores para Cada Lado

Você pode definir cores diferentes para cada lado da borda:

border-color: red green blue yellow;
/* Topo Direita Base Esquerda */

Ou definir separadamente:

border-top-color: red;
border-right-color: green;
border-bottom-color: blue;
border-left-color: yellow;

Exemplo Prático

.box {
border-style: solid; /* Necessário para exibir a borda */
border-width: 5px;
border-color: red; /* Define a cor da borda */
}

Bordas Individuais no CSS

O CSS permite definir bordas separadas para cada lado de um elemento usando as propriedades:

  • border-top
  • border-right
  • border-bottom
  • border-left

Cada uma dessas propriedades pode receber largura, estilo e cor.

Sintaxe

elemento {
border-top: largura estilo cor;
border-right: largura estilo cor;
border-bottom: largura estilo cor;
border-left: largura estilo cor;
}

Exemplo:

.caixa {
border-top: 2px solid red;
border-right: 4px dashed blue;
border-bottom: 3px dotted green;
border-left: 5px double black;
}

Valores Aceitos

Largura

Define a espessura da borda.

border-top-width: 5px;
border-right-width: thin;
border-bottom-width: medium;
border-left-width: thick;

Estilo

Define o estilo da borda.

border-top-style: solid;
border-right-style: dashed;
border-bottom-style: dotted;
border-left-style: double;

Cor

Define a cor da borda.

border-top-color: red;
border-right-color: blue;
border-bottom-color: green;
border-left-color: black;

Exemplo Completo

HTML

<div class="caixa">Exemplo de bordas individuais </div>

CSS

.caixa {
border-top: 5px solid red;
border-right: 4px dashed blue;
border-bottom: 3px dotted green;
border-left: 6px double black;
}

Bordas Arredondadas no CSS

A propriedade border-radius é usada para arredondar os cantos de um elemento.

Sintaxe

elemento {
border-radius: valor;
}

O valor pode ser em pixels (px), porcentagem (%) ou em/rem.

Exemplo Simples

HTML

<div class="caixa">Canto Arredondado</div>

CSS

.caixa {
border-radius: 10px;
background-color: lightblue;
padding: 20px;
}

Valores Possíveis

Todos os Cantos Iguais

border-radius: 20px;

Cantidades Diferentes por Canto

border-radius: 10px 20px 30px 40px;
/* topo-esquerda, topo-direita, baixo-direita, baixo-esquerda */

Valores Separados para X e Y

border-radius: 50px / 20px;
/* 50px no eixo X e 20px no eixo Y */

Arredondando Apenas Um Canto

border-top-left-radius: 15px;
border-top-right-radius: 30px;
border-bottom-right-radius: 45px;
border-bottom-left-radius: 60px;

Criando um Círculo

Se a largura e altura forem iguais, border-radius: 50% cria um círculo:

HTML

<div class="circulo">

CSS

.circulo {
width: 100px;
height: 100px;
background-color: red;
border-radius: 50%;
}

Border Image

O border-image no CSS permite usar uma imagem personalizada como borda de um elemento, ao invés de uma cor sólida ou gradiente.

Sintaxe do border-image

A propriedade border-image é definida assim:

.elemento {
border-image-source: url("borda.png"); /* Define a imagem */
border-image-slice: 30; /* Define a área da imagem usada como borda */
border-image-width: 10px; /* Define a largura da borda */
border-image-outset: 5px; /* Define a expansão da borda */
border-image-repeat: round; /* Define como a borda será repetida */
}

Ou usando a shorthand:

.elemento {
border-image: url("borda.png") 30 / 10px / 5px round;
}

Ordem da shorthand:
border-image: source slice / width / outset repeat;

Propriedades do border-image

border-image-source

Define a imagem usada como borda.

border-image-source: url("borda.png");

border-image-slice

Determina como a imagem será cortada em nove partes (4 cantos, 4 laterais e centro).

border-image-slice: 30; /* Divide a imagem a 30px da borda */
border-image-slice: 30% 20% 10% 5%; /* Definição personalizada (top, right, bottom, left) */

Se usar fill, o meio da imagem será usado como fundo do elemento.

border-image-slice: 30 fill;

border-image-width

Define a largura da borda.

border-image-width: 10px;

Pode ser definida em px, %, auto, ou valores relativos.

border-image-outset

Expande a borda para fora do elemento.

border-image-outset: 5px;

border-image-repeat

Controla como a imagem da borda será repetida.

Valor Descrição
stretch Estica a imagem para preencher a borda
repeat Repete a imagem ao longo da borda
round Repete, mas ajusta o tamanho para caber perfeitamente
space Como repeat, mas ajusta os espaços entre as repetições

border-image-repeat: round;

Exemplo Completo

.caixa {
border: 20px solid transparent;
border-image: url("borda.png") 30 / 20px / 10px repeat;
}

  • O border precisa estar definido para que border-image funcione.
  • Use transparent para evitar interferência de cor sólida na borda.
