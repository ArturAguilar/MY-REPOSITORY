Combinadores

Os combinadores em CSS são usados para selecionar e aplicar estilos a elementos com base na sua relação com outros elementos na estrutura do HTML. Existem diferentes tipos de combinadores que permitem selecionar elementos de forma mais específica.

Combinador de Descendente ( )

Seleciona elementos que estão dentro de um outro, independentemente de quão profundo o aninhamento seja.

div p {
color: red;
}

Explicação:
  • Aplica a cor vermelha a todos os parágrafos (p) dentro de um div.
  • O div p é o combinador de descendente.

Combinador de Filho Direto (>)

Seleciona elementos que são filhos diretos de um elemento pai específico.

div > p {
color: blue;
}

Explicação:

  • Aplica a cor azul apenas aos parágrafos (p) que são filhos diretos de um div.
  • Se um parágrafo estiver dentro de um div, mas não for um filho direto (por exemplo, se estiver dentro de outro elemento), ele não será selecionado.

Combinador de Irmão Adjacente (+)

Seleciona um elemento que é imediatamente precedido por outro elemento.

h1 + p {
color: green;
}

Explicação:

  • Aplica a cor verde ao primeiro parágrafo (p) que aparece logo após um h1.
  • Apenas o p diretamente após o h1 será afetado.

Combinador de Irmão Geral (~)

Seleciona todos os elementos que são irmãos de um determinado elemento.

h1 ~ p {
color: purple;
}

Explicação:

  • Aplica a cor roxa a todos os parágrafos (p) que são irmãos de um h1, ou seja, todos os p que estão na mesma hierarquia de contêiner que o h1.
