Pseudo-Classes

As pseudo-classes em CSS são usadas para definir o estilo de um elemento com base em seu estado ou condição, sem precisar modificar o HTML. Elas são extremamente úteis para alterar a aparência de um elemento em situações específicas, como quando um usuário passa o mouse sobre um link ou quando um formulário não é preenchido corretamente.

Pseudo-Classe :hover

Usada para aplicar estilos a um elemento quando o mouse passa sobre ele.

a:hover {
color: red;
}

Explicação:

  • Quando o usuário passa o mouse sobre o link (a), a cor do texto se torna vermelha.

Pseudo-Classe :focus

Aplicada a um elemento quando ele recebe o foco, como quando é clicado ou acessado via teclado.

input:focus {
border-color: green;
}

Explicação:

Quando um campo de entrada (input) recebe foco, a borda muda para verde.

Pseudo-Classe :active

Usada quando o elemento está sendo ativado (por exemplo, um link sendo clicado).

a:active {
color: blue;
}

Explicação:

  • Quando o link é clicado, a cor do texto se torna azul.

Pseudo-Classe :nth-child()

Seleciona um elemento com base em sua posição dentro de um contêiner.

ul li:nth-child(2) {
color: orange;
}

Explicação:

  • Aplique a cor laranja ao segundo item (li) da lista (ul).
  • O :nth-child() pode aceitar números, palavras-chave como odd (ímpar) e even (par), ou até expressões matemáticas, como 2n para todos os elementos em posições múltiplas de 2.

Pseudo-Classe :first-child

Seleciona o primeiro filho de um elemento.

ul li:first-child {
font-weight: bold;
}


Explicação:

  • Aplica um estilo de texto em negrito ao primeiro item (li) de uma lista (ul).

Pseudo-Classe :last-child

Seleciona o último filho de um elemento.

ul li:last-child {
color: purple;
}

Explicação:

  • Aplica a cor roxa ao último item (li) de uma lista (ul).

Pseudo-Classe :not()

Seleciona elementos que não correspondem a um seletor específico.

p:not(.special) {
color: gray;
}

Explicação:

  • Aplica a cor cinza a todos os parágrafos (p), exceto aqueles com a classe .special.
