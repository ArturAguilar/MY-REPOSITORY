Introdução aos Atributos de Formulários

Os atributos de formulários em HTML controlam o comportamento e a funcionalidade dos campos em um formulário. Estes atributos podem ser aplicados aos elementos de formulário como form, input, textarea, select, entre outros.

Atributos Comuns para o Elemento form

action

Define o local onde os dados do formulário serão enviados.

<form action="submit_form.asp">

method

Define o método de envio do formulário (GET ou POST).

  • GET: Envia dados na URL.
  • POST: Envia dados no corpo da requisição (mais seguro para grandes volumes de dados).

<form method="post">

target

Define onde o formulário será exibido após o envio (exemplo: em uma nova janela ou aba).

  • _self (padrão): Abre na mesma aba.
  • _blank: Abre em nova aba.
  • _parent: Abre no frame pai.
  • _top: Abre na janela principal.

<form target="_blank">

enctype

Especifica o tipo de codificação para o envio de formulários (usado quando o formulário contém arquivos).

<form enctype="multipart/form-data">

autocomplete

Ativa (on) ou desativa (off) a sugestão automática de dados.

<form action="processar.php" autocomplete="off">

novalidate

Remove a validação automática do formulário.

<form action="processar.php" novalidate>

Atributos Comuns para o Elemento input

type

Define o tipo de entrada (ex: texto, senha, email, etc.).

<input type="text">

value

Define o valor padrão do campo ou o valor que será enviado ao servidor quando o formulário for submetido.

<input type="text" value="João">

name

Define o nome do campo, que é usado para identificar os dados no servidor.

<input type="text" name="username">

placeholder

Exibe um texto temporário dentro do campo de entrada que desaparece quando o usuário começa a digitar.

<input type="text" placeholder="Digite seu nome">

required

Indica que o campo é obrigatório e não pode ser deixado em branco.

<input type="text" required>

readonly

Impede que o usuário edite o campo, mas permite que ele visualize seu conteúdo.

<input type="text" value="Texto fixo" readonly>

disabled

Desativa o campo, impedindo qualquer interação com ele.

<input type="text" disabled>

Atributos Comuns para o Elemento textarea

rows

Define a altura da área de texto em linhas.

<textarea rows="4"></textarea>

cols

Define a largura da área de texto em colunas.

<textarea cols="50"></textarea>

maxlength

Define o número máximo de caracteres que o usuário pode inserir.

<textarea maxlength="100"></textarea>

min e max

Define valores mínimos e máximos para números e datas.

<input type="number" min="1" max="100">

step

Define o intervalo entre valores permitidos.

<input type="number" min="0" max="10" step="2">

(Os valores possíveis são: 0, 2, 4, 6, 8, 10)

pattern

Define um padrão de entrada para validar o campo.

<input type="text" pattern="[A-Za-z]+" title="Apenas letras">

Atributos Comuns para o Elemento select

name

Define o nome do campo, que será usado para identificar os dados no servidor.

<select name="cars">

multiple

Permite que o usuário selecione várias opções.

<select name="cars" multiple>

size

Define o número de opções visíveis.

<select name="cars" size="5">

Atributos Específicos de Botões

type

Define o tipo do botão (submit, reset ou button).

<<button type="submit">Enviar</button>/p>

disabled

Desativa o botão.

<button type="submit" disabled>Enviar</button>

autofocus

Define que o campo ou botão terá o foco automaticamente ao carregar a página.

<input type="text" autofocus>
