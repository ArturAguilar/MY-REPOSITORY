Elementos de Formulário em HTML

Os elementos de formulário em HTML são usados para coletar informações do usuário. Eles são inseridos dentro da tag form e podem ser de diferentes tipos, como caixas de texto, botões, menus suspensos e muito mais.

O Elemento form

O elemento form serve como um contêiner para todos os elementos de entrada do formulário. Ele define como os dados serão enviados.

<form action="processar.php" method="post">
...
</form>

O Elemento input

O input é o elemento mais usado em formulários e pode ter diferentes tipos, como text, password, email, checkbox, radio, submit, entre outros.

<input type="text" name="nome" placeholder="Digite seu nome">

Principais tipos de input:

  • type="text" → Campo de texto simples.
  • type="password" → Campo de senha (esconde os caracteres digitados).
  • type="email" → Campo para e-mail (valida formato automaticamente).
  • type="checkbox" → Caixa de seleção.
  • type="radio" → Botão de opção (radio).
  • type="submit" → Botão para enviar o formulário.

<input type="radio" name="genero" value="masculino"> Masculino
<input type="radio" name="genero" value="feminino"> Feminino

O Elemento label

O label é usado para descrever um elemento de entrada, tornando a experiência do usuário mais acessível.

<label for="nome">Nome:</label>
<input type="text" id="nome" name="nome">

O atributo for no label deve corresponder ao id do campo de entrada.

O Elemento textarea

Usado para entrada de texto de múltiplas linhas.

<textarea name="mensagem" rows="4" cols="50"></textarea>

O Elemento select e option

Usado para criar um menu suspenso.

<select name="carro">
<option value="volvo">Volvo</option>
<option value="bmw">BMW</option>
<option value="audi">Audi</option>
</select>

Para permitir múltiplas seleções, use o atributo multiple:

<select name="carros" multiple>

O Elemento button

Cria um botão clicável dentro do formulário.

<button type="submit">Enviar</button>

O Elemento fieldset e legend

Usado para agrupar elementos do formulário.

<fieldset>
<legend>Informações Pessoais</legend>
<label for="nome">Nome:</label>
<input type="text" id="nome" name="nome">
</fieldset>

O Elemento datalist

Cria uma lista de sugestões para um campo de entrada.

<input list="cidades" name="cidade">
<datalist id="cidades">
<option value="São Paulo">
<option value="Rio de Janeiro">
<option value="Belo Horizonte">
</datalist>
