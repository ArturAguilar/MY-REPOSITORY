Atributos Relacionados ao Formulário em Elementos HTML

Esses atributos permitem conectar elementos de entrada (input, select, textarea, etc.) a um formulário específico, mesmo que não estejam dentro dele.

form (Associa o Campo a um Formulário)

Conecta um elemento a um formulário usando o id do formulário.

<form id="meuFormulario" action="processar.php">
<input type="text" name="nome">
</form>

<input type="text" name="email" form="meuFormulario">

✅ O campo "email" será enviado junto com o formulário, mesmo estando fora dele.

formaction (URL de Envio Específica para um Botão)

Define um destino de envio diferente apenas para botões de envio (input type="submit" e button type="submit").

<form action="pagina_padrao.php">
<input type="submit" value="Enviar Padrão">
<input type="submit" value="Enviar Alternativo" formaction="outra_pagina.php">
</form>

✅ O primeiro botão envia para "pagina_padrao.php", o segundo para "outra_pagina.php".

formenctype (Tipo de Codificação de Dados)

Define o tipo de codificação dos dados enviados (usado apenas com method="post").

Opções:

  • application/x-www-form-urlencoded (Padrão)
  • multipart/form-data (Necessário para upload de arquivos)
  • text/plain (Envio como texto simples)

<form action="upload.php" method="post">
<input type="file" name="arquivo">
<input type="submit" value="Upload Normal">
<input type="submit" value="Upload Arquivo" formenctype="multipart/form-data">
</form>

✅ O segundo botão permite o envio de arquivos corretamente.

formmethod (Método de Envio para um Botão)

Permite que um botão altere o método de envio do formulário (GET ou POST).

<form action="processar.php" method="post">
<input type="submit" value="Enviar com POST">
<input type="submit" value="Enviar com GET" formmethod="get">
</form>

✅ O primeiro botão usa POST, o segundo força o envio via GET.

formtarget (Destino da Resposta)

Controla onde abrir a resposta da submissão do formulário.

Valores:

  • _self (Padrão) → Mesma aba.
  • _blank → Nova aba.
  • _parent → Janela pai.
  • _top → Janela principal.

<form action="processar.php">
<input type="submit" value="Abrir na mesma aba">
<input type="submit" value="Abrir em nova aba" formtarget="_blank">
</form>

✅ O segundo botão abrirá a resposta em uma nova aba.

formnovalidate (Ignora Validações)

Desativa a validação de campos apenas para um botão de envio.

<form action="processar.php">
<input type="email" required placeholder="Digite seu e-mail">
<input type="submit" value="Enviar Normal">
<input type="submit" value="Ignorar Validação" formnovalidate>
</form>

✅ O segundo botão permite enviar o formulário sem preencher o campo de e-mail.
