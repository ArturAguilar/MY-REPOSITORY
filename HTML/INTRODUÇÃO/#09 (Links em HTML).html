Links em HTML

Links são criados usando a tag <a> (anchor).

Eles são usados para navegar entre páginas, redirecionar para outros sites, baixar arquivos ou executar ações.

Sintaxe Básica

<a href="URL do destino">Texto ou conteúdo clicável</a>

  • O atributo href especifica o endereço do link.
  • O texto entre a abertura e o fechamento da tag <a> é exibido como o link clicável.

Exemplo

<a href="https://www.exemplo.com">Visite o Exemplo</a>

Tipos de links

Links para outra página

<a href="https://www.google.com">Acesse o Google</a>

Links para um e-mail

Use o prefixo mailto: no atributo href.

<a href="mailto:exemplo@email.com">Envie um e-mail</a>

Links para a mesma página

Use #id para criar âncoras para seções específicas da página.

<a href="#section1">Ir para a Seção 1</a>
<h2 id="section1">Seção 1</h2>

Links para download

O atributo download permite que um arquivo seja baixado em vez de aberto.

<a href="arquivo.pdf" download>Baixe o arquivo</a>

Atributos Importantes

target

Define com o link será aberto.

  • _self (padrão): Abre na mesma janela/aba.
  • _blank: Abre em uma nova aba ou janela.

<a href="https://www.exemplo.com" target="_blank">Abrir em nova aba</a>

rel

Especifica a relação entre o documento atual e o link.

Exemplo: nofollow (para links que não devem passar SEO).

<a href="https://www.exemplo.com" rel="nofollow">Link sem SEO</a>

title

Fornece uma descrição extra ao passar o mouse sobre o link.

<a href="https://www.exemplo.com" title="Acesse o site Exemplo">Exemplo</a>

Estilização de Links

Links podem ser estilizados com CSS usando diferentes pseudo-classes:

  • :link → Link ainda não visitado.
  • :visited → Link já visitado.
  • :hover → Quando o cursor está sobre o link
  • :active → Quando o link é clicado.

Exemplo

<style>
a {
text-decoration: none; /* Remove o sublinhado */
color: blue;
}
a:hover {
color: red; /* Muda a cor ao passar o mouse */
}
</style>
<a href="https://www.exemplo.com">Link Estilizado</a>
