Listas em HTML

Tipos de Listas em HTML

Lista Ordenada (<ol> - Ordered List)

  • Usada para itens que precisam seguir uma ordem específica.
  • Por padrão, os itens aparecem numerados.

Exemplo:

<ol>
<li>HTML</li>
<li>CSS</li>
<li>JavaScript</li>
</ol>

Lista Não Ordenada (<ul> - Unordered List)

  • Usada para itens sem uma ordem específica.
  • Por padrão, os itens aparecem com marcadores (bullets).

Exemplo:

<ul>
<li>Café</li>
<li>Chá</li>
<li>Suco</li>
</ul>

Lista de Definições (dl - Definition List)

  • Usada para termos e suas descrições.
  • Composta por: <dt>: Define o termo. <dd>: Define a descrição.

Exemplo:

<dl>
<dt>HTML</dt>
<dd>Linguagem de marcação usada para criar páginas web.</dd>

<dt>CSS</dt>
<dd>Linguagem de estilo usada para definir a aparência de páginas web.</dd>
</dl>

Listas Aninhadas

Listas podem ser aninhadas, ou seja, uma lista dentro de outra.

Exemplo:

<ul>
<li>Front-End
<ul>
<li>HTML</li>
<li>CSS</li>
<li>JavaScript</li>
</ul>
</li>
<li>Back-End
<ul>
<li>Node.js</li>
<li>Python</li>
</ul>
</li>
</ul>

Personalização de Listas

Alterando os Estilos de Marcadores (Bullets)

Pode-se usar a propriedade CSS list-style-type para personalizar marcadores.

Exemplo:

<style>
ul {
list-style-type: square; /*
Marcador em forma de quadrado */
}
</style>

<ul>
<li>Item 1</li>
<li>Item 2</li>
</ul>

Outros valores de list-style-type:

  • circle
  • square
  • none (sem marcadores)

Alterando o Estilo de Números em Listas Ordenadas

Use o atributo type no <ol> .

Exemplo:

<ol type="A">
<li>Item 1</li>
<li>Item 2</li>
</ol>

Valores do atributo type:

  • 1: Números (padrão).
  • A: Letras maiúsculas.
  • a: Letras minúsculas.
  • I: Números romanos maiúsculos.
  • i: Números romanos minúsculos.

Começando uma Lista em um Número Específico

Use o atributo start no <ol>.

Exemplo:

<ol start="5">
<li>Item 5</li>
<li>Item 6</li>
</ol>

Personalização com CSS

<style>
ol.custom {
list-style-type: upper-roman; /*
Números romanos maiúsculos */
padding-left: 20px;
color: #d70000;
}
</style>

<ol class="custom">
<li>Primeiro</li>
<li>Segundo</li>
</ol>
