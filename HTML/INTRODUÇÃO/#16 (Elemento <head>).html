O Que é o head?

  • Definição: O elemento contém metadados sobre o documento HTML. Esses metadados não são exibidos diretamente na página, mas são essenciais para o funcionamento e configuração dela.
  • Posição: O está localizado no início do documento HTML, antes do .

Estrutura Básica do head

<html>
<head>
<title>Título da Página</title>
<meta charset="UTF-8">
<meta name="description" content="Descrição da

página">
<meta name="keywords" content="HTML, CSS, JavaScript">
<meta name="author" content="Seu Nome">
<link rel="stylesheet" href="estilo.css">
<script src="script.js"></script>
</head>
<body>
Conteúdo da página aqui.
</body>
</html>

Elementos Comuns no head

title

  • Define o título exibido na aba do navegador.

<title>Minha Página</title>

meta

  • Define metadados do documento, como codificação de caracteres, descrição e palavras-chave.

<meta charset="UTF-8">
<meta name="description" content="Descrição da página">
<meta name="keywords" content="HTML, CSS, JavaScript">
<meta name="author" content="Seu Nome">
<meta name="viewport" content="width=device-width, initial-scale=1.0">

link

  • Conecta recursos externos, como arquivos CSS.

<link rel="stylesheet" href="estilo.css">

script

  • Adiciona ou referencia arquivos JavaScript.

<script src="script.js"></script>

style

  • Define estilos CSS diretamente na página.

<style>
body {
font-family: Arial, sans-serif;
}
</style>

base

  • Define a URL base para links e caminhos relativos.

<base href="https://www.exemplo.com/">

Atributos Úteis para SEO e Responsividade

Meta Description:

  • Melhora o SEO ao fornecer uma descrição da página para os motores de busca.

<meta name="description" content="Descrição breve da página">

Palavras-chave:

  • Ajuda os motores de busca a identificar os tópicos da página.

<meta name="keywords" content="HTML, tutorial, aprendizado">

Viewport:

  • Torna a página responsiva, ajustando-a para dispositivos móveis.li>

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Exemplo Completo de um head Otimizado

<!DOCTYPE html>
<html lang="pt-br">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="description" content="Aprenda HTML com este tutorial completo e fácil de entender.">
<meta name="keywords" content="HTML, tutorial, programação web">
<meta name="author" content="Artur Aguilar">
<title>Tutorial de HTML</title>
<link rel="stylesheet" href="estilo.css">
<script src="script.js"></script>
</head>
<body>
Conteúdo da página.
</body>
</html>
