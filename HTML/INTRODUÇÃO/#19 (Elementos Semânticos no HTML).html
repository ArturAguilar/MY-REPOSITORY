Elementos Semânticos no HTML

Elementos semânticos são aqueles que possuem um significado claro, tanto para desenvolvedores quanto para navegadores e mecanismos de busca. Eles ajudam a estruturar o conteúdo da página de forma mais compreensível.

Principais Elementos Semânticos do HTML5

header

  • Representa o cabeçalho de uma página ou seção.
  • Geralmente contém logotipos, títulos, menus de navegação, etc.

<header>
<h1>Bem-vindo ao Meu Site</h1>
<nav>
<a href="#home">Home</a> |
<a href="#about">Sobre</a> |
<a href="#contact">Contato</a>
</nav>
</header>

nav

  • Define uma seção de links de navegação.

<nav>
<ul>
<li><a href="#home">Home</a></li>
<li><a href="#services">Serviços</a></li>
<li><a href="#contact">Contato</a></li>
</ul>
</nav>

section

  • Agrupa conteúdo relacionado, como capítulos ou seções de um documento.

<section>
<h2>Sobre Nós</h2>
<p>Somos uma empresa especializada em tecnologia.</p>
</section>

article

  • Representa um conteúdo independente e autocontido, como postagens de blogs, artigos de notícias, etc.

<article>
<h2>Última Notícia</h2>
<p>Hoje lançamos um novo produto incrível!</p>
</article>

aside

  • Define conteúdo relacionado ou complementar, como barras laterais, citações ou anúncios.

<aside>
<h3>Aviso Importante</h3>
<p>Estamos em manutenção no dia 01/02.</p>
</aside>

footer

  • Representa o rodapé de uma página ou seção.
  • Contém informações como direitos autorais, links de política de privacidade, etc.

<footer>
<p>© 2025 Meu Site. Todos os direitos reservados.</p>
</footer>

main

  • Define o conteúdo principal de uma página.
  • Deve conter o conteúdo mais relevante, excluindo cabeçalhos, rodapés, menus de navegação, etc.

<main>
<h1>Bem-vindo ao Meu Site</h1>
<p>Aqui você encontrará informações úteis sobre nossos serviços.</p>
</main>

figure e figcaption

  • figure: Usado para agrupar mídia (imagens, diagramas, etc.) com legendas.
  • figcaption: Define a legenda de um elemento figure.

<figure>
<img src="imagem.jpg" alt="Descrição da imagem">
<figcaption>Figura 1: Uma bela paisagem.</figcaption>
</figure>

mark

  • Destaca texto importante.

<p>Este é um <mark>texto destacado</mark>.</p>

Exemplo Completo de Uso

<!DOCTYPE html>
<html lang="pt-br">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Exemplo de Elementos Semânticos</title>
</head>
<body>
<header>
<h1>Meu Site</h1>
<nav>
<ul>
<li><a href="#home">Home</a></li>
<li><a href="#about">Sobre</a></li>
<li><a href="#contact">Contato</a></li>
</ul>
</nav>
</header>

<main>
<section id="about">
<h2>Sobre Nós</h2>
<p>Somos apaixonados por tecnologia e inovação.</p>
</section>

<article> <h2>Última Notícia</h2> <p>Nosso novo produto já está disponível no mercado.</p> </article>

<aside>
<h3>Aviso Importante</h3>
<p>Manutenção agendada para 01/02.</p>
</aside>
</main>

<footer>
<p>© 2025 Meu Site. Todos os direitos reservados.</p>
</footer>
</body>
</html>
