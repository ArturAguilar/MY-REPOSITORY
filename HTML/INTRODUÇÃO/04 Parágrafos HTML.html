Parágrafos HTML

O que são parágrafos HTML?

  • O elemento <p> é usado para definir um parágrafo de texto.
  • O texto dentro do parágrafo será exibido com espaçamento acima e abaixo por padrão.
  • Ideal para organizar o conteúdo textual na página.

Exemplo de Parágrafo

<p>Este é um parágrafo de exemplo. O parágrafo pode ter várias linhas, mas o navegador automaticamente cria quebras de linha ao final de cada parágrafo.</p>

Quebras de Linha

O navegador não faz quebras de linha automáticas dentro de um parágrafo, exceto no final de cada <p>.

Para adicionar uma nova linha dentro de um parágrafo, utilize a tag <br> (break line):

<p>Primeira linha.<br>Segunda linha.</p>

Espaçamento

O elemento <p> tem espaçamento automático entre os parágrafos, o que ajuda a organizar o conteúdo de forma legível.

Boas Práticas

Use <p> para dividir grandes blocos de texto em seções menores.

Não use <p> para elementos que não sejam parágrafos, como listas ou cabeçalhos.
