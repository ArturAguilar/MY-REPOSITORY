Tudo sobre imagens em HTML

Imagens são incorporadas em páginas HTML usando a tag <img>.

Esta tag é autocontida, ou seja, não possui uma tag de fechamento.

Imagens em HTML

Sintaxe Básica

<img src="URL da imagem" alt="Descrição alternativa">

  • src: (Source) Especifica o caminho ou URL da imagem.
  • alt: (Alternative text) Fornece um texto alternativo que será exibido caso a imagem não carregue ou para acessibilidade.

Exemplo

<img src="imagem.jpg" alt="Descrição da imagem">

Atributos Comuns

width e height

Especificam a largura e altura da imagem (em pixels ou porcentagem).

<img src="imagem.jpg" alt="Imagem redimensionada" width="300" height="200">

title

Adiciona um texto exibido ao passar o mouse sobre a imagem.

<img src="imagem.jpg" alt="Imagem" title="Texto ao passar o mouse">

style

Permite estilizar diretamente a imagem com CSS.

<img src="imagem.jpg" alt="Imagem estilizada" style="border: 2px solid red;">

Formatos Comuns de Imagens

  • JPEG/JPG: Ideal para fotos e imagens com muitas cores.
  • PNG: Suporta transparência; indicado para gráficos e ícones.
  • GIF: Usado para animações.
  • SVG: Imagem vethttps://www.w3schools.com/html/html_tables.asp orial escalável; ideal para gráficos e ícones.
  • WebP: Formato moderno com alta compressão e qualidade.

Imagens Responsivas

Para que uma imagem se ajuste dinamicamente ao tamanho da tela, use o estilo:

<img src="imagem.jpg" alt="Imagem responsiva" style="width: 100%; height: auto;">

Usando Imagens como Links

Uma imagem pode ser usada como link, colocando-a dentro de uma tag <a>:

<a href="https://www.exemplo.com">
<img src="imagem.jpg" alt="Imagem como link">
</a>

Imagens em Diretórios Diferentes

Mesma pasta

<img src="imagem.jpg" alt="Imagem na mesma pasta">

Subpasta

<img src="subpasta/imagem.jpg" alt="Imagem em subpasta">

Pasta anterior

<img src="../imagem.jpg" alt="Imagem em pasta anterior">

Imagens Externas

Use URLs completas para imagens hospedadas em outros sites.

<img src="https://www.exemplo.com/imagem.jpg" alt="Imagem externa">

Exemplo Completo

<!DOCTYPE html>
<html>
<head>
<title>Exemplo de Imagens</title>
</head>
<body>

<h1>Imagens em HTML</h1>

<!-- Imagem básica -->
<img src="imagem.jpg" alt="Descrição da imagem" width="300">

<!-- Imagem como link -->
<a href="https://www.google.com">
<img src="imagem.jpg" alt="Imagem como link" width="300">

</a>
<!-- Imagem responsiva -->
<img src="imagem.jpg" alt="Imagem responsiva" style="width: 100%; height: auto;">

</body>
</html>

Mapas de Imagem em HTML

  • Um mapa de imagem permite definir áreas clicáveis em diferentes partes de uma imagem.
  • É utilizado quando uma única imagem precisa conter vários links.

Sintaxe Básica

  • Use a tag <img> com o atributo usemap para associar a imagem a um mapa.
  • Crie o mapa com a tag <map>.
  • Dentro do mapa, defina áreas clicáveis com a tag <area>.

<img src="imagem.jpg" usemap="#mapa" alt="Mapa de imagem">

<map name="mapa">
<area shape="rect" coords="x1,y1,x2,y2" href="link1.html" alt="Descrição do Link 1">
<area shape="circle" coords="x,y,raio" href="link2.html" alt="Descrição do Link 2">
<area shape="poly" coords="x1,y1,x2,y2,x3,y3" href="link3.html" alt="Descrição do Link 3">
</map>

Elementos Principais

<img> com usemap

  • O atributo usemap conecta a imagem ao mapa.
  • O valor de usemap deve começar com um # seguido do nome do mapa.

<map>

  • Define o mapa de imagem.
  • Use o atributo name para identificá-lo.

<area>

  • Define as áreas clicáveis no mapa.
  • Principais atributos:
    • shape: Define o formato da área (retângulo, círculo ou polígono).
    • coords: Coordenadas da área.
    • href: URL ou link associado à área.
    • alt: Texto alternativo para acessibilidade.

Formas e Coordenadas

Retângulo (rect)

Coordenadas: x1,y1,x2,y2 (canto superior esquerdo e inferior direito).

<area shape="rect" coords="34,44,270,350" href="link1.html" alt="Área retangular">

Círculo (circle)

Coordenadas: x,y,raio (centro do círculo e raio).

<area shape="circle" coords="180,150,50" href="link2.html" alt="Área circular">

Polígono (poly)

Coordenadas: Sequência de pontos x1,y1,x2,y2,... (os vértices do polígono).

<area shape="poly" coords="140,20,180,80,220,20" href="link3.html" alt="Área poligonal">

Exemplo Completo

<!DOCTYPE html>
<html>
<head>
<title>Mapa de Imagem</title>
</head>
<body>

<h1>Exemplo de Mapa de Imagem</h1>

<!-- Imagem com mapa associado -->
<img src="imagem.jpg" usemap="#mapaExemplo" alt="Mapa de imagem" style="width: 500px;">

<!-- Mapa de imagem -->
<map name="mapaExemplo">
<area shape="rect" coords="50,50,200,200" href="https://www.link1.com" alt="Link Retângulo">
<area shape="circle" coords="300,150,75" href="https://www.link2.com" alt="Link Círculo">
<area shape="poly" coords="400,50,450,150,400,250" href="https://www.link3.com" alt="Link Polígono">
</map>

</body>
</html>

Imagens de Fundo em HTML

  • Para adicionar uma imagem como fundo, utilizamos o CSS, pois o HTML não possui uma tag específica para isso.
  • O estilo é aplicado através do atributo style ou em uma folha de estilo externa.

Adicionando uma Imagem de Fundo

Usando o atributo style diretamente em HTML

<body style="background-image: url('imagem.jpg');">

Usando uma folha de estilo CSS

body {
background-image: url('imagem.jpg');
}

Controle da Imagem de Fundo

Repetição da imagem (background-repeat)

  • repeat: Repetição padrão.
  • no-repeat: Não repete.
  • repeat-x: Repete horizontalmente.
  • repeat-y: Repete verticalmente.

<body style="background-image: url('imagem.jpg'); background-repeat: no-repeat;">

Posicionamento da imagem(background-position)

  • Define onde a imagem será posicionada.
  • Valores comuns: top, center, bottom, left, right, ou coordenadas específicas (x% y% ou xpx ypx).

<body style="background-image: url('imagem.jpg'); background-position: center;">

Tamanho da imagem (background-size)

  • Ajusta o tamanho da imagem de fun
  • Valores:
    • auto: Tamanho padrão.
    • cover: A imagem cobre todo o elemento, mantendo a proporção.
    • contain: A imagem é redimensionada para caber no elemento.
    • Dimensões específicas: width height (em px, %, etc.).

<body style="background-image: url('imagem.jpg'); background-size: cover;">

Fixação da imagem (background-attachment)

  • scroll: A imagem rola junto com a página.
  • fixed: A imagem permanece fixa ao rolar.

<body style="background-image: url('imagem.jpg'); background-attachment: fixed;">

Exemplo Completo

<!DOCTYPE html>
<html>
<head>
<title>Exemplo de Fundo com Imagem</title>
<style>
body {
background-image: url('fundo.jpg');
background-repeat: no-repeat;
background-position: center;
background-size: cover;
background-attachment: fixed;
}
</style>
</head>
<body>

<h1>Bem-vindo!</h1>
<p>Este é um exemplo de uma página com imagem de fundo.</p>

</body>
</html>

O Elemento <picture> em HTML

O elemento <picture> é usado para fornecer versões alternativas de uma imagem para dispositivos ou situações específicas, como diferentes tamanhos de tela ou resoluções.

Sintaxe Básica

<picture>
<source srcset="imagem-grande.jpg" media="(min-width: 800px)">
<source srcset="imagem-media.jpg" media="(min-width: 400px)">
<img src="imagem-pequena.jpg" alt="Descrição da imagem">
</picture>

Descrição dos Elementos:

  • <picture>
    • Contém diferentes fontes de imagem.
    • Permite que o navegador escolha a mais apropriada.
  • <source>
    • Define uma fonte de imagem alternativa.
    • Atributos:
      • Contém diferentes fontes de imagem.
      • Permite que o navegador escolha a mais apropriada.
      • srcset: Caminho para a imagem.
      • media: Define condições, como largura de tela.
  • <img>
    • Contém diferentes fontes de imagem.
    • Permite que o navegador escolha a mais apropriada.

Atributos Importantes


srcset:

  • Define um ou mais arquivos de imagem.
  • Exemplo com múltiplas resoluções:

<source srcset="imagem1.jpg 1x, imagem2.jpg 2x">

media:

  • Especifica a condição para usar a imagem.
  • Utiliza media queries CSS:
    • (min-width: 600px)
    • (max-width: 400px)

<source srcset="imagem.jpg" media="(min-width: 600px)">

type:

  • Define o formato da imagem (exemplo: image/webp, image/jpeg).

<source srcset="imagem.webp" type="image/webp">

Exemplo Completo

<!DOCTYPE html>
<html>
<head>
<title>Exemplo do Elemento <picture></title>
</head>
<body>

<h1>Exemplo de Imagens Responsivas</h1>

<picture> <source srcset="grande.jpg" media="(min-width: 800px)">
<source srcset="media.jpg" media="(min-width: 400px)">
<source srcset="pequena.webp" type="image/webp">
<img src="fallback.jpg" alt="Imagem de exemplo">
</picture>

<p>Esta imagem muda dependendo do tamanho da tela e do formato suportado.</p>

</body>
</html>
