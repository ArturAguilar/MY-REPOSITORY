O Que é o <div>?

  • Definição: O elemento
    é um contêiner de bloco usado para agrupar elementos HTML, permitindo aplicar estilos ou manipular conteúdo como uma unidade.
  • Tipo: Elemento de bloco (block-level).

Quando Usar <div>?

  • Para organizar a estrutura de uma página web.
  • Para aplicar estilos com CSS a um grupo de elementos.
  • Para facilitar a manipulação de elementos com JavaScript.

Exemplo Básico de Uso:

<div>
<h1>Título</h1>
<p>Este é um parágrafo dentro de um <div>.</p>
</div>

Estilizando o <div> com CSS:

O <div> não possui estilos padrão, mas pode ser personalizado com CSS.

Exemplo:

<style>
div {
background-color: #d70000; /* Cor de fundo */
color: white; /* Cor do texto */
padding: 20px; /* Espaçamento interno */
margin: 10px; /* Espaçamento externo */
border: 2px solid black; /* Borda */
}
</style>

<div>
Este é um div estilizado.
</div>

Usando Classes e IDs com <div>

  • Classes: Para aplicar o mesmo estilo a múltiplos <div>.
  • IDs: Para estilizar ou manipular um
    específico.

Exemplo com Classes:

<style>
.caixa {
background-color: lightblue;
padding: 15px;
border: 1px solid gray;
}
</style>

<div class="caixa">Div 1</div>
<div class="caixa">Div 2</div>

Exemplo com ID:

<style>
#unico {
background-color: yellow;
font-weight: bold;
}
</style>

<div id="unico">Div único estilizado</div>

Nesting: Div Dentro de Div

<div> pode conter outros <div> para organizar o layout.

Exemplo:

<div style="background-color: lightgrey;">
<div style="background-color: white; margin: 10px;">
Conteúdo interno.
</div>
</div>

Quando Evitar <div>?

Não use <div> desnecessariamente quando outros elementos semânticos (como header, footer, section, etc.) forem mais apropriados para descrever o conteúdo.
