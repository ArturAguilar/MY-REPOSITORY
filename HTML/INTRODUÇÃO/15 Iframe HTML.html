O Que é o iframe?

  • Definição: O elemento iframe é usado para incorporar outro documento HTML dentro de uma página.
  • Significado: "iframe" significa Inline Frame.
  • Tipo: Elemento inline.

Exemplo Básico:

<iframe src="https://www.w3schools.com" width="600" height="400">
Seu navegador não suporta iframes.
</iframe>

  • src: Especifica o URL do documento que será carregado dentro do iframe
  • width e height: Controlam o tamanho do iframe (em pixels ou porcentagem).

Atributos Comuns do iframe

src:

  • Define o URL da página a ser exibida.

<iframe src="https://example.com"></iframe>

width e height:

  • Especificam a largura e altura do iframe.

<iframe src="https://example.com" width="800" height="600"></iframe>

title:

  • Adiciona uma descrição acessível para o iframe (importante para acessibilidade).

<iframe src="https://example.com" title="Exemplo de Iframe"></iframe>

allowfullscreen:

  • Permite que o iframe seja exibido em tela cheia.

<iframe src="https://example.com" allowfullscreen></iframe>

name:

  • Nomeia o iframe para referência em links ou scripts.

<iframe src="https://example.com" name="meuIframe"></iframe>

sandbox:

  • Restringe ações dentro do iframe para aumentar a segurança.

<iframe src="https://example.com" sandbox></iframe>

  • allow-forms: Permite envio de formulários.
  • allow-same-origin: Permite acesso a recursos na mesma origem.
  • allow-scripts: Permite execução de scripts (sem acesso ao conteúdo).

<iframe src="https://example.com" sandbox="allow-scripts allow-same-origin"></iframe>

Exemplo de Navegação com target

Um link pode carregar uma página dentro de um iframe usando o atributo target.

<a href="https://www.w3schools.com" target="meuIframe">Abrir no iframe</a>
