Entidades HTML

Entidades HTML são usadas para representar caracteres especiais que, de outra forma, poderiam ser interpretados como código HTML. Elas são úteis para exibir símbolos, caracteres reservados ou caracteres que não podem ser facilmente digitados no teclado.

Por que usar entidades HTML?

Alguns caracteres têm funções específicas no HTML, como:

  • < e > (usados para definir tags HTML)
  • & (usado para especificar outras entidades)
  • " e ' (para aspas duplas e simples)

Se esses caracteres forem usados diretamente no HTML sem entidades, podem causar erros no código ou serem interpretados de maneira errada pelo navegador.

Emojis em HTML

Os emojis são representados em HTML por códigos Unicode. Eles podem ser usados diretamente no código HTML, CSS ou JavaScript para exibir ícones gráficos que representam emoções, objetos, símbolos e muito mais.

Como Usar Emojis em HTML

Usando o próprio emoji

Basta copiar e colar o emoji diretamente no código:

<p>😃 Olá, mundo!</p>

Usando códigos Unicode

O Unicode dos emojis pode ser usado com &#x seguido do código hexadecimal:

<p>😀 Isso é um emoji!</p>

O código & #x1F600; representa 😀 (rosto sorridente).
